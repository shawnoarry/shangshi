# 《破晓南烟》（暂定名）剧本大纲

**【故事核心】** 
以南洋兄弟烟草公司在清末民初的崛起为主线，展现民族资本家在帝国主义夹缝中“实业救国”的艰难历程，以及在历史洪流中，中国早期产业工人觉醒与抗争的壮阔画卷。

## 序幕：风起南洋，星火初燃 (1904 - 1911)
*   **情节点 1：立志与初创。** 简照南痛感清朝腐朽，华人在海外备受屈辱，连公海航行执照都无法领取。他抱定“实业救国”之心，1904年在上海印制“清末美女牌九”画片，成为国产香烟画片鼻祖。
*   **情节点 2：暗中输血。** 1909-1910年，南烟虽资金不宽裕，简照南仍毫不犹豫拿出巨款，派族伯简昌达赴香港，亲手交予陈炯明，资助孙中山的革命事业。
*   **小高潮：** 1911年辛亥革命爆发，海外华侨受鼓舞，国货畅销。“双喜”“飞马”等品牌迅速打破洋烟垄断，简照南加冕“烟草大王”。

## 第一幕：商海狂飙，抵御外患 (1912 - 1922)
**（核心冲突：民族企业与跨国巨头的生死博弈）**

*   **情节点 1：广而告之。** 民国初年，南烟设立罕见的“广告部”，聘请潘达微等专业人士，首创“中国人吸中国烟”的口号。广告打遍全国国货展览会、酒馆青楼，将商业营销提升至爱国主义宣传的高度。
*   **情节点 2：官方背书。** 1916年，孙中山出席南烟与金星保险的联合大会，发表盛赞演说，定性南烟为“为国家挽回利权者”，将南烟的地位推向顶峰。简照南因热心捐资助学（复旦、南开等）和建孤儿院，被尊为“商界师表，南洋菩萨”。
*   **高潮戏：拒降洋人。** 随着南烟壮大，跨国烟草巨头视其为心腹大患，接连在1914年、1917年、1918年、1922年发起吞并企图。
    *   *【高光台词与名场面】* 1917年，面对洋人“占股百分之六十”的优厚合并诱惑，简玉阶当面反驳：“名为合并，实则无异战败国之屈降……若一旦屈降外人，纵不为社会唾骂，亦令提倡国货者灰心。而我公司营业必从此失败矣！”谈判决裂，保住了“民族品牌”的灵魂。

## 第二幕：阵痛与撕裂，赤色风暴 (1922 - 1924)
**（核心冲突：劳资矛盾爆发，无产阶级觉醒）**

*   **情节点 1：繁华背后的阴影。** 随着企业大扩张，为了追求利润，南烟厂方开始制定严苛的30余条管理条款，取消病假工资，并无故开除工人。昔日的“民族英雄”在此刻展露出了资本家压迫的一面。
*   **情节点 2：夜校里的火种。** 1922年，向警予来到上海沪东地区，在女工密集的南洋烟厂秘密开办工人夜校。她亲自授课，启发女工（如唐景星、陈倩如）向资本家作斗争。1924年7月，上海烟草行业第一个党小组在南烟成立。
*   **高潮戏：七千人大罢工。** 1924年9月9日，忍无可忍的7000余名南烟工人爆发大罢工。李立三到上海协调救援。
    *   *【动作名场面：血战百老汇】* 勇猛的工人大哥顾顺章代表工人前往百老汇路的公司总部抗议，遭资本家保镖毒打，后被租界巡捕打得遍体鳞伤奄奄一息抓进巡捕房（后被党组织营救入党）。
    *   *【视觉名场面：口红标语】* 向警予教不识字的女工们，用女人的口红，在南烟的香烟盒上写下“不自由毋宁死”。这些写着血字的烟盒，成了轰动上海滩的反抗符号。
    *   *【舆论战】* 毛泽东化名“子任”发表文章，痛批《民国日报》歪曲罢工真相。广州、香港跨地域响应，没收南烟香烟拍卖救济上海工人。

## 第三幕：同仇敌忾，淬火成钢 (1925)
**（核心冲突：民族大义面前的冰释前嫌与共同对外）**

*   **情节点 1：五卅惨案的血洗。** 1925年五卅惨案爆发，帝国主义的屠刀挥向中国人。刚刚经历过罢工撕裂的南烟厂，在民族存亡面前发生了心态的转变。
*   **情节点 2：阶级联合。** 向警予亲自带队上街游行，南洋烟厂3000多工人成为主力。为了支援全国罢工，南烟工人选择“加班做工”，将10万元加班费全部捐给学生和罢工工人。
*   **情节点 3：勿忘国耻。** 厂方（资本家）与工人在此刻达成了某种精神上的和解与共鸣。此后每年的5月30日，南洋烟草公司用自家产品拼成“五卅”字样，并附文提醒国民“勿忘国货烟，勿忘国耻”。

## 尾声：长路漫漫，实业不熄 (引出抗战)
*   **旁白/字幕交代（基于第107页资料缩写）：** 时代的洪流滚滚向前。伴随着代代南烟人栉风沐雨的坚定信念，当抗日战争全面爆发时，南烟公司汉口制造厂毅然再次迁往重庆，成为西南烟草工业最重要的一员，为中国经济发展在废墟中保存了最珍贵的实力。
*   **定格画面：** 滚滚长江上，满载南烟机器的船只向着大西南驶去；画面叠化回当年女工们用口红写在烟盒上的“不自由毋宁死”。全剧终。
