# 《火种》第二幕剧情梳理 v0.3

整理时间：2026-05-30  
版本定位：根据最新反馈，对第二幕中“五四时期身份受质疑”相关内容做安全调整，作为后续大纲修订和台本展开的现行参考。

补充说明：当前主线相关口径，后续以 `新提交/0530南洋烟草开场情景剧《火种》剧情大纲3.0.docx` 与 `新提交/0530第二轮修改简述.docx` 为最新标准；本文件负责把其中第二幕的安全边界和展开逻辑写清。

## 一、本轮调整原则

最新反馈已明确：五四时期身份受质疑的部分完全隐去，相关内容不进入剧情、不进入台词、不进入提报说明的正面表述。

因此，第二幕中原先围绕“质疑南烟身份”“敬告国人”“公开自证”“脱离日籍”“以公开账目回应攻击”等一组动作，全部不再使用。

这并不意味着第二幕中段失去情绪抬升。改法应当是：

把原先的“身份风波”段，改写为“时代风潮上扬、企业责任加重、社会声浪进入工厂”的段落，让第二幕仍然能够从商业发展顺利抬高到时代责任，再进入第三轮谈判的高潮。

## 二、哪些内容要完全删去

以下内容在后续大纲、台本、提报说明中均不再触碰：

1. “国货身份遭质疑”“身份风波”“日资攻击”“日籍问题”等表述。  
2. “敬告国人”“公开自证”“脱离日籍手续”等动作。  
3. 以报纸质询、商会追问、门口等说法等方式构成的身份辩白戏。  
4. 以“公开账目、公开招股、改组公司”作为回应身份危机的舞台因果。

其中，“公开招股”“股东名册”即使在史实上与后续发展有关，现阶段也不建议继续作为第二幕中段的主戏动作使用，以免再次把叙事带回敏感段。

## 三、这一段应该怎么改

核心改法不是把原段抽空，而是把原来的“质疑—回应”结构，改成“风潮—担当—拔高”结构。

### 1. 第二轮谈判之后，不再进入“身份辩白”

原来如果写成：

- 报纸递到桌上  
- 外界发问  
- 简氏兄弟开门给国人看  
- 员工翻账  
- 起草“敬告国人”  

这一整套都应删去。

改为：

第二轮谈判结束后，舞台不再落到一场解释和辩白上，而是直接转入国货风潮继续升高、市场与社会压力同步加重的阶段。报童喊出的不再是对南烟的质问，而是上海街头正在升高的“提倡国货、振兴实业、社会自强”之声。柜台上的双喜、红双喜不再只是买卖货物，而是在这一风潮中被越来越多的人当作中国人自强的一部分。商号来信、催货单、加急订单一份份压上桌，说明南烟承受的已经不只是市场机会，更是时代寄予它的分量。

### 2. 用“责任加重”代替“身份回应”

第二幕中段的重心，应从“你是谁”改为“你做大以后，准备怎样承担”。

可以把这段写成三组更稳妥、也更可演的动作：

1. 国货风潮推高市场声势。  
报童、买主、烟贩、商号伙计在不同光区完成一个清楚的变化：双喜从柜台下走到台面上，从一地问货变成多地催货。  

2. 社会责任压到桌上。  
1920年的北方旱灾和上海米荒依次压上长桌。董事会定下“每售一箱，捐洋五元，香烟价目并不增加”的办法；南烟又采购越南米到沪，以平粜和发米票接济贫民。义学灯光、留学名单随后进入桌面。简氏兄弟讨论的已不只是价格、渠道和工厂，也包括企业在灾年里怎样出钱、出米、出力，怎样把实业所得的一部分回到教育与民生。  

结合 0530 新提交版，这一段可再具体半步：货栈一角，简照南带员工清点赈济米粮、核对出货单，受资助的青年学生代表前来致谢，并愿意为赈灾出力。简照南以“学之不兴，实业难兴”回应，把教育、实业与时代责任连到同一现场里。  

3. 工厂里的时代声浪开始变大。  
报童带来烟草行业工人风潮消息，上海厂的员工和青年职员开始谈论工人的处境、做事的体面和时代的变化。这里不写成直接政治口号，也不把南烟写成运动的领导者，而是写成时代声音已经进入工厂空间。

### 2.5 已采用的背景音：1916年孙中山对南烟的评价

0530 新提交版已经采用这一处理：把1916年孙中山出席大会时对南烟“为国家挽回利权者也”的评价，放在第一轮谈判之后、第二轮谈判之前的发展段中，以报童一句短促吆喝带出。

这一处理的好处在于：

1. 时间上成立。  
1914年第一次收买之后，南烟继续扩张市场、设立分销机构、筹建上海厂；1916年的这段评价正落在第一轮与第二轮之间。  

2. 功能上合适。  
它不必作为正面人物对话出现，只需作为报童转述、会场回声、留声机片段或屏幕字幕后的画外音，成为市场做大、国货声势抬高的一道历史回声。新提交版采用的正是这一轻处理方式。  

3. 位置上要克制。  
这句不宜放在过场一里，因为过场一主要承接辛亥前后到1914年前后的市场起势；若提前放入，会把1916年的信息压进第二幕起点之前。更稳的做法，是让它出现在第一轮谈判结束后的发展蒙太奇中，与上海分公司、渠道铺开、工厂推进等信息并列。  

建议使用方式：

- 报童短促带出“孙文盛赞南洋烟草，为国家挽回利权者也”；  
- 同时配合市场地图点亮、订单上桌、分销与上海厂推进；  
- 声音一过即收，不延长成革命叙述主线。  

### 3. 第三轮谈判前的“桌面证据”要换掉

原来如果第三轮谈判前，桌上压的是：

- 公开账目  
- 招股文件  
- 股东名册  
- 身份回应材料  

现在建议改成：

- 订单簿  
- 员工名册  
- 机器清单  
- 上海厂与渠道资料  
- 1920赈济账本  
- 助学或办学名单

这样第三轮谈判的立意仍然成立，而且更集中：

简照南守住的不再是一场辩白后的名义，而是一门已经连着柜台、工厂、员工、学生和灾民的中国实业。

## 四、第二幕中段的新版结构

### 第五段：国货风潮延续、1920赈济义举与社会责任

时间：1919至1920年前后。  
核心作用：把第二幕从商业发展抬高到时代责任，但不进入身份风波。

建议写法：

第二轮谈判后的僵持，没有把舞台带向解释与辩白，而是带向更大的时代风潮。1919至1920年前后，上海街头的空气变了。报童喊出的是提倡国货、振兴实业、社会自强的消息；柜台上的双喜被摆得更正，买主问货更急，商号来信更密。南烟此时面对的，不只是如何把烟卖出去，而是怎样让这门中国人办起来的实业，经得起时代的期待。

紧接着，1920年北方数省大旱与上海米荒相继压上长桌。南烟董事会定下“每售一箱，捐洋五元，香烟价目并不增加”的办法；随后又采购越南米到沪，以平粜和发米票接济贫民。货栈一角，简照南带着员工清点赈济米粮、核对出货单，受资助的青年学生代表前来致谢，并愿为赈灾出力。简照南答“学之不兴，实业难兴”，把办厂、救急和育人放到同一条线上。第二幕的情绪由此从商战走向担当。

### 第六段：工人觉醒声浪与第三轮谈判

时间：1921至1922年初。  
核心作用：把社会声浪推到第三轮谈判前。

建议写法：

当南烟的市场、工厂和社会责任越拉越开，时代的另一种声音也走进厂房。报童和青年职员带来的是烟草行业工人风潮的消息，工厂里开始有人谈起工人的处境、做事的体面和新的时代空气。这样的声浪不直接构成谈判的因果，却让第三轮谈判的分量陡然不同：桌上已经不只是牌号和机器，更有工人饭碗、企业信誉和中国人自己办实业的根。

1922年初，遮佛士来华，第三轮谈判回到长桌。此时桌上的关键不宜再围绕“证明自己是谁”，而应围绕“已经做起来的这门实业还能不能由中国人自己作主”。外部画外音越强调优厚条件，简照南越要把视线落回南烟已经承担起来的现实：市场、员工、工厂、教育，以及1920年灾年中已经兑现过的责任。第三轮的立场因此更硬，情绪也更足。

### 第七段：兄弟交接仍然保留

兄弟精神交接这一段不需要大改，只需把交接的内容从“经过身份风波后的守业”调整为“经过市场扩张、社会责任和时代风潮后的守业”。

简照南交给简玉阶的，不再是一轮风波后的解释权，而是：

- 白鹤失败后重新站起的那团火  
- 双喜和红双喜走上柜台后的国货分量  
- 员工、机器和工厂一路做大的现实责任  
- 教育、1920赈济义举和时代风潮压到企业身上的那份担当

## 五、对台本和提报的直接影响

### 台本层

第二幕中不再写以下具体场面：

- 门口等说法  
- 报纸直接质问南烟身份  
- 起草“敬告国人”  
- 当众翻账自证  
- 用脱离日籍、公开自证等动作完成一场解释戏

可以直接改写成：

- 报童报出时代风潮  
- 双喜与订单一起被推到更亮处  
- 1920年北方旱灾、上海米荒与教育事项压上桌  
- 工人声浪从厂房里起来  
- 第三轮谈判因此更具时代重量

### 提报层

后续对上说明时，这一段建议统一表述为：

“第二轮谈判后，第二幕不再展开敏感身份风波，而是转入国货风潮持续升高、1920年北方旱灾与上海米荒所带来的社会责任，以及工人觉醒声浪。这样既避开敏感内容，也能把南烟从商业发展自然推进到时代担当，再把第三轮谈判抬到更高的立意上。”

## 六、建议一句话梗概

第二幕以双喜、红双喜的发展为入口，把第一、第二次谈判嵌入南烟走向上海和市场扩张的过程中；第二轮之后，不写身份风波，改写为国货风潮延续、1920年北方旱灾与上海米荒引出的赈济和教育责任，以及工人觉醒声浪进入工厂；最终在1922年第三轮谈判中，把“这门中国人办起来的实业还能不能由中国人自己作主”的问题推到最高处，并完成简照南向简玉阶的精神交接。
