# docs 文档使用索引

> 用途：作为 `docs/` 的唯一入口文件，说明当前目录结构、各类文档功用，以及后续写作/核验时应该怎么用。  
> 更新日期：2026-05-26

## 一、目录结构

当前 `docs/` 已按功用拆成八类，根目录只保留本索引：

```text
docs/
├─ 文档使用索引.md
├─ 00_项目说明/
├─ 01_会议纪要/
├─ 02_执行稿/
├─ 03_资料索引/
├─ 04_方案储备/
├─ 05_文稿工程/
├─ 06_新增资料/
└─ 灵感素材库/
```

分类原则：

- `00_项目说明/`：仓库资料、同步、OCR 说明。
- `01_会议纪要/`：会议原文、会议记录整理、最新口径、任务拆解。
- `02_执行稿/`：当前或曾经进入执行讨论的节目稿、大纲、规划。
- `03_资料索引/`：《上实百廿》和上实发展资料的事实、情节、时间线索引。
- `04_方案储备/`：母题、结构、创意方向，作为写作和替换方案储备。
- `05_文稿工程/`：后续正式撰写、迭代、提交和归档的剧本稿件工程区。
- `06_新增资料/`：后续新增资料的原始文件、待整理摘录和已整理资料卡。
- `灵感素材库/`：早期灵感稿、AI试写稿和已降权结构，只作表达参考，不作当前执行口径或史实依据。

## 二、优先阅读路径

第一次接手项目，建议按这个顺序读：

1. `00_项目说明/工作流程与资料取用规则.md`
   先确认当前项目定位、资料取用优先级、依赖与 skill 状态，避免把南洋烟草段落误当整台活动总纲。

2. `00_项目说明/materials.md`
   先了解原始资料、OCR 文本、Git LFS 和重新生成 OCR 的方法。

3. `01_会议纪要/20260523导演会会议记录工作汇总.md`
   了解最新导演会口径：肖恩/编剧团队负责什么、南洋烟草节目已达成什么共识、后续有哪些执行约束。

4. `02_执行稿/南洋烟草开场情景剧剧情大纲指导稿.md`
   当前最接近后续剧本创作的工作稿，是继续拆正式大纲、分场台本、多媒体脚本和舞蹈衔接的主入口。

5. `05_文稿工程/01_剧本大纲/20260525_南洋烟草开场情景剧_剧情大纲_提交细化版.md`
   在提交参考版基础上进一步对齐过往项目提交版细致度，适合用于对上沟通、导演讨论和下一步台本拆分。

6. `05_文稿工程/文稿工程说明.md`
   开始写正式剧本、大纲、分场台本或准备提交版本时，先看这里确认稿件放置、中文文档要求和版本命名规则。

7. `03_资料索引/上实百廿全书情节索引.md`、`03_资料索引/上实百廿候选节目素材池.md`、`03_资料索引/上实百廿标志性情节点时间轴.md`
   需要补事实、换素材、查事件位置时，再进入这三份资料索引。

## 三、各目录说明

### 00_项目说明

#### `00_项目说明/materials.md`

说明本项目哪些原始 PDF 通过 Git LFS 管理，哪些 OCR 文本已纳入仓库同步，以及如何重新生成 OCR。

适用场景：

- 新设备克隆项目。
- 查哪些文件是原始资料、哪些是可检索文本。
- 需要重新跑 OCR 或重建页面图片。

使用提醒：

- 这是仓库资料说明，不是节目内容索引。
- OCR 文本可检索但不等于最终史实，正式引用仍需回看原 PDF 页。

#### `00_项目说明/工作流程与资料取用规则.md`

统一当前仓库的资料取用优先级、目录职责、文稿流转规则、事实核验流程，以及依赖和 skill 状态。

适用场景：

- 新设备或新协作者接手项目。
- 判断旧稿、灵感稿、执行稿和正式大纲的优先级。
- 确认南洋烟草段落与整台45周年活动之间的边界。
- 查当前项目是否需要新增依赖或 skill。

使用提醒：

- 这是仓库工作流规则，不是剧本内容稿。
- 如用户或导演组给出新口径，应先更新本文件或在会议纪要中补充，再进入文稿修改。

### 01_会议纪要

#### `01_会议纪要/00_新转写待整理/`

新会议、采访、沟通录音的自动转写或未整理记录先放在这里。

适用场景：

- 新增会议转写还没整理成纪要。
- 采访转写需要先存档、后续再拆成事实素材或创作口径。
- 需要保留原始转写，方便追溯会议原话。

使用提醒：

- 建议命名为 `YYYYMMDD_会议主题_原始转写.docx` 或 `YYYYMMDD_采访主题_原始转写.md`。
- 整理完成后，在 `01_会议纪要/` 新建正式整理稿，并在本索引补充。
- 如采访中出现可写入剧本的事实素材，应转入 `06_新增资料/` 或补入 `03_资料索引/`。

#### `01_会议纪要/20260523205847-转写_佳。的快速会议-逐字稿文本-1.docx`

导演会原始自动转写文档，是 `20260523导演会会议记录工作汇总.md` 的来源文件。

适用场景：

- 需要回看会议原话、判断讨论语境。
- 核对整理稿是否遗漏某个口径。
- 追溯具体发言人和时间点。

使用提醒：

- 这是自动转写原文，存在识别误差和口语噪声。
- 不建议直接作为对外稿件引用；应先整理、复核再使用。

#### `01_会议纪要/20260523导演会会议记录工作汇总.md`

根据导演会逐字稿整理出的工作汇总，是当前创作口径的重要依据。

主要内容：

- 整台节目结构的新口径。
- 肖恩/编剧团队需要负责或介入的工作。
- 南洋烟草开场节目已达成共识的框架。
- 演员、排练、多媒体、史实核验等执行信息。
- 上实小镇、音乐剧串联、童声合唱等补充事项。

适用场景：

- 判断当前创作方向是否符合导演会共识。
- 确认哪些任务和肖恩团队有关。
- 写任务清单、排期、沟通备忘。

使用提醒：

- 这是会议口径，不是最终剧本。
- 逐字稿为自动转写，专名、年份、个别表述需复核。

### 02_执行稿

#### `02_执行稿/南洋烟草开场情景剧剧情大纲指导稿.md`

当前最重要的执行稿。它把导演会共识扩展成“引子 + 第一幕 + 幕间一 + 第二幕 + 幕间二 + 第三幕 + 尾声转舞蹈”的完整剧情框架。

主要内容：

- 整体创作判断。
- 建议时长与结构。
- 核心人物、贯穿物件和舞台系统。
- 每段的叙事功能、初步故事逻辑、素材对照、承上启下。
- 尾声如何过渡到舞蹈演绎。
- 多媒体、台词、舞台动作的分工。
- 后续正式剧本大纲的拆分建议。

适用场景：

- 写正式剧本大纲。
- 拆分场台本。
- 和导演、多媒体、舞蹈编导沟通。
- 判断每一幕是否有完整故事逻辑。

使用提醒：

- 后续台本应优先从此稿继续深化。
- 稿内史实页码来自 OCR 和索引，进入正式台词前必须回看 `画册扫描.pdf` 原页精校。

#### `02_执行稿/南洋烟草情景剧初步大纲.md`

早期初步大纲，提出“东山再起、外商并购、战时内迁”三段式结构。

适用场景：

- 回看最初的三段结构如何形成。
- 查早期对多媒体快剪、物件母题和资料充分性的判断。

使用提醒：

- 本稿仍按较早的 10 至 12 分钟口径写成。
- 最新创作应以 `02_执行稿/南洋烟草开场情景剧剧情大纲指导稿.md` 和 `01_会议纪要/20260523导演会会议记录工作汇总.md` 为准。

#### `02_执行稿/南洋烟草情景剧专项规划.md`

对南洋烟草情景剧的节目定位、内容边界、候选方案和写作原则做了专项规划。

适用场景：

- 判断南洋烟草段落在整台节目中承担什么功能。
- 避免把南洋烟草情景剧写成整台活动总纲。
- 回看早期结构候选和写作边界。

使用提醒：

- 是规划性文件，不是最新剧情执行稿。
- 可作为导演阐述和方案说明的辅助资料。

### 03_资料索引

#### `03_资料索引/上实百廿分段内容概述.md`

按《上实百廿》章节顺序整理全书内容，帮助理解每章主题、事件和节目价值。

适用场景：

- 初步读懂《上实百廿》的结构。
- 判断某个章节适合写成舞台戏、视频、串词还是背景。
- 写导演阐述或资料说明时快速回顾章节逻辑。

使用提醒：

- 适合顺读理解，不适合快速查某个具体情节点。
- 具体事实仍需结合 OCR 或原 PDF 精校。

#### `03_资料索引/上实发展脉络与上实百廿情节点梳理.md`

把上实集团发展脉络和《上实百廿》的关键情节点放在一起，建立“百年南烟”和“上实45周年”的连接。

适用场景：

- 说明为什么南洋烟草能服务上实45周年主题。
- 查“120年的火种，45年的承接”这条表达逻辑。
- 写上实与南烟、香港与上海、祖业与绿色健康之间的关联。

使用提醒：

- 这是早期综合底稿，后续拆出了更细的时间轴、素材池和情节索引。
- 如果要查具体情节，优先看 `03_资料索引/上实百廿全书情节索引.md`。

#### `03_资料索引/上实百廿全书情节索引.md`

把 `画册扫描.pdf` 的全书内容整理成可检索、可复用的情节地图。

适用场景：

- 查某个事件在哪一章、可怎么舞台化。
- 查人物、物件、母题对应哪些章节。
- 为剧本寻找替换素材。

使用提醒：

- 这是内容索引，不是总目录。
- 进入正式剧本或串词前，需要对入选事实回看 PDF 原页精校。

#### `03_资料索引/上实百廿候选节目素材池.md`

把《上实百廿》中可舞台化、可视频化、可串词化的素材拆成候选池。

主要内容：

- A 类：适合舞台演绎。
- B 类：适合多媒体快剪。
- C 类：适合台词补充或背景串联。
- 当前情景剧可替换结构。
- 事实校对优先级。

适用场景：

- 需要替换某一幕事件。
- 需要给多媒体找画面素材。
- 需要判断哪些情节适合演员演、哪些只适合字幕或视频。

使用提醒：

- 后续写作时，应先从素材池挑选“主演绎事件”和“多媒体补充事件”，再做史实核验。

#### `03_资料索引/上实百廿标志性情节点时间轴.md`

按时间顺序整理南烟及上实相关关键节点，并为每个节点提供舞台抓手。

适用场景：

- 检查剧情时间顺序。
- 写幕间视频或年份字幕。
- 需要把一个事件放回历史前后关系中。

使用提醒：

- 适合查时间线和转场逻辑。
- 不承担完整事实校对功能。

- 祖业
- 机器
- 账本
- 校舍

- 给多媒体、舞美、舞蹈找意象。

适用场景：

- 比较不同结构方案。
- 当前三幕结构需要调整或替换时，找备用结构。
- 写方案提案时说明为何选择当前结构。

使用提醒：

- 本文件保留大量灵感储备，不是当前最终执行方案。
- 最新执行方案以 `02_执行稿/南洋烟草开场情景剧剧情大纲指导稿.md` 为准。

### 05_文稿工程

#### `05_文稿工程/文稿工程说明.md`

说明后续剧本稿件工程区的目录分工、命名规则和版本流转方式。

适用场景：

- 新建正式剧本大纲。
- 拆分场台本或台词稿。
- 准备对外提交版本。
- 归档已被替代的旧稿。

使用提醒：

- 新稿件优先进入 `05_文稿工程/`，不要继续堆在 `02_执行稿/`。

- 文件名建议保留日期、节目名、稿件类型和版本号。
- 当前文稿工作统一使用中文文档：文件名、标题、正文、版本说明和修改记录均使用中文；必要英文专名、原文引用、软件路径和扩展名除外。

#### `05_文稿工程/01_剧本大纲/`

存放正式剧情大纲、幕场结构、人物关系和主线梳理。

#### `05_文稿工程/03_正式稿件/`

#### `05_文稿工程/04_版本归档/`

### 06_新增资料

#### `06_新增资料/新增资料说明.md`

说明后续新增资料从原始文件到整理入索引的流转方式。

适用场景：

- 新收到 PDF、Word、图片、表格、音视频清单。
- 新资料完成 OCR 或人工摘录，但尚未核验。
- 新资料已经整理成事实卡、页码表或图片素材清单。

使用提醒：

- 新资料不要直接放根目录；先进入 `06_新增资料/01_原始资料/`。
- 提取文字或临时整理成果放 `06_新增资料/02_待整理资料/`。
- 核验完成后放 `06_新增资料/03_已整理入索引/`。
- 长期会反复调用的事实素材，应同步补入 `03_资料索引/` 相关文件。
- 后续新建资料说明、摘录、资料卡和索引表统一使用中文文件名、中文标题和中文正文。
