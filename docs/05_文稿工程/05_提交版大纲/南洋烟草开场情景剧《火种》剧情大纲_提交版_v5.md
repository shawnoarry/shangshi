# 南洋烟草开场情景剧《火种》

## 剧情大纲

### 核心主旨

本剧以南洋烟草早期发展中的三次关键危机为主线，讲述简氏兄弟和一代民族实业人在失败、诱惑与战火中守住民族工业主体性的故事。作品聚焦上实45周年特别活动中的一个语言类节目段落，用南洋烟草的祖业故事为整场活动点起一束“火”：实业报国落在具体行动里，落在机器、品牌、员工、信用和希望被一次次护下来的选择里。

全剧围绕三个问题推进：败了，还办吗；卖了，根还在吗；毁了，还能重新开机吗。三次追问层层加压，形成从经营失败、资本吞并到战时内迁的戏剧递进。舞台上，火、机器、账本和合同贯穿始终，让“国货”“实业”“祖业”成为可见、可听、可被人物承担的舞台动作。

### 剧情概述

1905年前后，年轻的简照南、简玉阶兄弟在香港旧货仓点起民族烟草工业的第一缕火。白鹤牌初有起色，却因商标纠纷遭遇重创。巡理府前，价值2000多元的香烟被当众销毁，一个刚被街面信任的民族品牌也被压入低谷。简照南承下这份羞辱，把工人、烟贩和买主的信任记进账本。

失败之后，兄弟陷入自责和挫败。简铭石以现实支持托住他们，工人和街面民众也以谨慎而真切的方式返场，给出对国货再起的期待。简氏兄弟由此选择东山再起。

随着南洋烟草在国货热潮、渠道扩张和上海工厂建设中逐渐壮大，外资巨头多次抛出收买与合并条件。第二幕以三次谈判为骨架，在谈判之间穿插南烟发展的舞台蒙太奇和多媒体快剪：订单增加、广告铺开、上海设厂、资本扩张、公开招股、工人增多。南烟越长大，谈判桌上的诱惑和压力越重。简氏兄弟在资金、扩厂、工资和市场风险之间守住企业主体性，让“国货”从一句口号变成必须承担的信用。

抗战时期，战火逼近，汉口厂抢救式搬迁，机器和物资沿长江西迁。第三幕按由轻到重的层次推进：前两段用汉口拆机和江上转运铺开现实压力，第三段把重心落在重庆厂战火中开工。人物在重开工时完成精神统一，把南烟自己的企业命运升到战时中国守住实业、守住家国希望的层面。情景剧在机器声重新响起时完成自身结尾；后续歌舞接住这份情绪，转入百年祖业与上实45年承接的昂扬表达。

### 整体风格与结构

本剧为历史情景剧，整体气质热烈、紧凑、厚重。舞台表达采用“现实场景 + 心理空间 + 舞台蒙太奇 + 多媒体信息层”的组合方式。外部压力通过画外音、字幕、合同文本、报纸标题、警报声和冰屏影像呈现；舞台正面聚焦简氏兄弟、家族支撑、工人群像、街面民众和企业内部选择。

全剧按12至15分钟体量组织，并与后半歌舞组成开场组合节目。结构为“引子 + 三幕 + 两段过场 + 尾声转歌舞”：

- 引子：白鹤初燃，巡理府前当众销毁，形成第一记冲击。
- 第一幕：从心理坠落到东山再起，回答“败了，还办吗”。
- 过场一：国货热潮、渠道扩张、上海升级，说明南烟为何成为对手。
- 第二幕：三轮并购谈判与两段内嵌蒙太奇交替推进，回答“越做大，越能卖吗”。
- 过场二：多媒体呈现南烟继续发展，报童轻动作带出战时新闻，商战压力转为战争压力。
- 第三幕：战时内迁、江上转运与重庆重开工，回答“毁了，什么必须留下”。
- 尾声：情景剧完成火种落点，再以短桥段承接后半歌舞。

### 出场人物

1. 简照南：史料中有明确原型的主要人物。作为全剧主行动人物，他前期有闯劲和实业理想，失败后有自责与动摇，第二幕在现实压力与民族主体性之间完成成熟选择。

2. 简玉阶：史料中有明确原型的主要人物。作为兄弟关系中的情绪力量，他更直接、更不服输，承担第二幕中守住根基的强烈表达。

3. 简铭石：史料中有明确原型的主要人物。作为第一幕精神支点，他提供现实支持，也把兄弟从失败情绪中拉回行动。

4. 账房/档案员：有资料基础的复合角色，贯穿账本、债务、资产、设备清册和档案传承。第一幕清点债务和机器物料，第二幕把账本、员工名册或公开账目放上谈判桌，第三幕转为设备清册和搬迁清单。

5. 工人代表与工人群像：有资料基础的复合角色，按不同年代呈现为多名工人代表。第一幕代表初创工人的失业焦虑，第二幕代表员工饭碗和工厂命运，第三幕成为战时拆机、搬运、重开工的行动力量。

6. 董事/股东复合角色：第二幕内部冲突人物，分出主和、主战、务实三种声音，代表资金、扩厂、工资和市场风险等现实考量。

7. 环境群像：报童、烟贩、商号伙计、商会代表、学生代表、广告宣传员、街市商贩、搬运工、船工、重庆厂工人等，承担时代背景、市场流动、国货气氛、战时转运和歌舞衔接。

舞台主要角色以简氏家族为核心。资料中另有明确姓名的历史人物，在后续台本中弱化为合同署名、报纸字幕、画外音或背景信息。环境戏和舞台蒙太奇中的配角以身份指代，舞台重心始终落在简氏家族和南洋烟草三次危机上。

### 场景设置

1. 香港码头/罗素街旧货仓外场：用于引子。市井声、搬货声、报童声和洋烟广告共同构成初创环境。

2. 巡理府前/白鹤销毁空间：用于引子的情绪转折。法槌声、外商画外音、字幕和火色灯光共同形成第一次重压。

3. 心理压力空间：用于第一幕前半段。冰屏、报纸标题、重叠人声和四面方位调度构成挫败感。

4. 清算仓库/账房空间：用于第一幕后半段。旧机器、账本、烟标、牌匾、物料箱、空烟盒和旧订单构成主要道具。

5. 国货发展蒙太奇空间：用于过场一和第二幕谈判间隔。演员调度、地图点亮、广告画翻转、烟箱传递、订单堆叠、工厂剪影共同呈现企业做大。

6. 谈判桌：用于第二幕。同一张桌子串联多轮并购谈判，通过合同厚度、灯光方向、人物站位和服装细节表现时间推进。

7. 战时工厂/码头/江面/重庆厂：用于第三幕。机器架、木箱、纤绳、船板、警报声和烟尘灯光完成多空间转换。

8. 重庆厂重开工/档案光：用于尾声。机器声完成情景剧的精神落点，搬迁清单转为档案信息，短桥段把情绪交给后半歌舞。

### 剧情大纲

#### 一、引子 ——【白鹤初燃】

舞台从香港码头或罗素街旧货仓外的热闹场景开始。汽笛、叫卖、搬货、报童声交织，年轻的简氏兄弟穿行其间。新做好的“白鹤”烟标被工人举起，烟贩说街面上有人开始问这个牌子。简玉阶兴奋，简照南望向远处洋烟广告，意识到真正的对手还在前方。

兄弟短对话点出初心：洋烟一船一船进来，中国人的钱一船一船出去，中国人自己造、自己卖、自己被国人信任，这件事有了超出生意的分量。舞台呈现初创的朝气与街面的信任。

氛围突然转冷。外商指控以画外音进入，冰屏出现法庭、商标、败诉、销毁等纪录片式字幕。巡理府前，工人本能护住烟箱，烟贩怕被牵连，简玉阶上前理论；简照南拦住他，把烟贩的欠账记进账本，又从工人手中接过一包“白鹤”，放进被勒令销毁的箱内。他站在烟箱前承下这一次羞辱。刚才那句“让国人信任”的理想，在这一刻变成他对工人、烟贩和街面买主的欠账与责任。舞台以火色写意映出白鹤标识的崩塌，字幕明确为“价值2000多元‘白鹤’牌香烟在港英当局巡理府前当众销毁”。

引子结束时，火色仍在。简照南把残留的一张白鹤标识夹进账本，简玉阶看着空了的烟箱，灰烬中的微光接入第一幕。

#### 二、第一幕 ——【败后再起】

第一幕承接引子末尾的火色和账本。简照南将残留的白鹤标识夹进账本后，舞台随即进入抽象心理空间。四面声音压上来，外商声音、报纸标题、烟贩低语、工人追问、街面议论彼此叠加。冰屏反复出现“败诉”“仿冒”“销毁”“清算”“谁还敢买”“谁还敢卖”等词。简玉阶几次冲向声音来源，又被一圈圈人影拦回；简照南抱账本站在中央，所有声音像一笔笔追来的债。

心理空间中，兄弟的挫败感逐渐具象化。简玉阶的不甘外放为一声声“凭什么”；简照南的压力内收为工人明天的饭碗、烟贩躲开柜面的眼神、街面买主被牵连后的沉默。前面“让国人信任”的理想倒转为最重的压力：信任一旦被打掉，受伤的是一个牌号，也是所有愿意试一试国货的人。

现实清算随后显形。账房报出机器、物料、债务和无人敢买的局面，工人守在旧机器旁，等一个明日去处。简铭石上场，把他们从心理空间拉回仓库现实。他先问机器在哪里、物料在哪里、债怎么还、人还剩几个，再以现实支持和精神提醒推动兄弟重新站起。简铭石的托底落在资金、资源和信念三处：败一次可痛，自己认输才真正断路。

街面民众在后半段返场。引子中的烟贩、商号伙计、报童和买主从心理空间的质疑声中走出，先是犹豫地把空烟盒、赊账条或旧订单放回桌上，随后有人低声说“等你们再开张，我还摆出来”，也有人说“这次没敢摆在柜面，下次你们把牌子做稳，我第一个问货”。这些声音带着市井里的谨慎、同情和不服气，与前半段“谁还敢买、谁还敢卖”的质疑形成对照。

这一幕由精神坠落走向重新开机。账房在账本上划掉旧债、写下新章程，简铭石把现实支撑放上桌，工人掀开蒙住机器的布，烟贩和商号伙计把旧订单压在账本旁。钱、章程、印章、工具和订单落在同一张桌上。旧机器重新启动，机器声从断续到稳定，第一束暖光亮起。

#### 三、过场一 ——【火怎样烧起来】

机器声延续，转为国货热潮的节拍。报童举起国货新闻，商号递来订单，烟箱在演员手中传递，地图上点亮南洋、广州、上海、南京、镇江、苏州等节点。演员推转长桌，旧仓空间变成销售桌、广告桌和工厂桌，高阳大楼与上海制造厂剪影出现。

过场一完成从重新站起到第一次被外资盯上的逻辑。南烟靠产品、渠道、广告和公众信任做出声势。越来越亮的地图和广告被报价数字覆盖。外商画外音从“指控”变为“合作”，长桌被推到舞台中央，进入第二幕第一轮谈判。

#### 四、第二幕 ——【和战之间】

同一张谈判桌串起三轮并购谈判，三轮之间留出两段发展蒙太奇。每一次谈判结束，舞台都短暂离开桌面，用舞台蒙太奇和多媒体共同呈现南烟在时间间隔中的发展：订单增加、渠道外扩、上海厂建设、广告传播、公开招股、工人队伍扩大。南烟越发展，外商的报价越高，内部的现实压力也越重。

第一轮，外商抛出收买条件，带着威胁也带着诱惑。南烟刚刚站稳，简照南以高价回应，拖住对方，也显示这门买卖没有廉价退路。董事担心资金，简玉阶压住怒气，账房把现有资产和债务放到桌上。第一轮落在弱势中的机智。

第一段内嵌蒙太奇随即展开。谈判桌灯光压暗，报童从台口穿过，报出国货声势和各地订单。多媒体上，地图从香港、南洋继续点亮到广州、上海、南京、镇江、苏州；商号伙计把订单一张张递给账房，工人推过烟箱，广告宣传员翻出新的烟画。舞台动作简洁，重点信息交给多媒体和节奏快剪。外商画外音在快剪末尾再次出现：销量、渠道、工厂、品牌都被他们看见了。

第二轮，外方提出更完整的合并条件。条件从单纯收买转向资产、品牌、经营权和市场控制。内部分歧被推到台前：联合带来资金和安全，守住主体性意味着继续面对巨头压力和未知风险。简照南面对扩厂、工资、债务和市场，简玉阶承担更直接的情绪表达。工人代表第一次进入谈判边缘，问的是“厂还归谁，明天听谁开机”。

第二段内嵌蒙太奇从“做大”转向“做大后的压力”。多媒体出现高阳大楼、上海制造厂、广告烟画、公开账目、招股信息和员工名册；报童声音从兴奋变得急促，商号订单与资金缺口交替出现。舞台上，工人数量增多，机器节拍更密，账房把账页越翻越快。南烟的声势越盛，桌上的合同越厚。蒙太奇末尾，打字机声压过机器声，第三轮谈判的厚合同被放到桌中央。

第三轮，合同更厚，条件更诱人。外商声音提出资产合并、品牌保留、仍由简氏经营等看似周全的方案。董事/股东复合角色提出现实焦虑：厂要扩，薪要发，债要还，市场还在打。工人代表把员工名册、机器清单和账本放到桌上：名字留下，制造权和未来交出去，南烟的根就被抽走了。简照南合上合同，报价纸被账本压住，机器声重新盖过打字机声。

第二幕落在现实压力中的承担：南烟缺钱，根不能丢；国货活得艰难，主体性仍要守住。三轮谈判层层加码，把南烟越做越大之后面对的清醒选择推到台前：卖掉更容易，守住更艰难。

#### 五、过场二 ——【时代压下来】

第二幕合上的合同留在桌上，报童带着新报纸从侧台跑过。多媒体先延续南烟这一时期的发展：新厂、订单、广告、招股、员工队伍、国货声势继续滚动。舞台动作保持轻量，报童换报、账房翻页、工人从远处推过一只烟箱，人物正面戏暂时收住。

屏幕信息逐渐转冷。商业新闻被九一八、淞沪、卢沟桥、武汉告急、民族工业内迁等战时信息取代。报童第二次跑回舞台时，手中的报纸已是战报。灯光由暖转青灰，远处警报声进入。账房合上商业账本，打开设备清册；战时负责人走到桌前，只留一句：“账以后再算，机器今天要走。”第三幕从这句话进入。

#### 六、第三幕 ——【江上护火】

第三幕分为三段，前两段铺垫压力，第三段集中完成高潮。汉口拆机和江上转运负责把现实困难摆出来：机器重、船少、时间紧、人心乱。重庆厂战火中开工成为主戏，前两段积累的焦虑和疑问都在这一段得到回答。

第一段为汉口拆机。厂内一面维持生产，一面拆卸设备。市面上仍缺烟，工人担心停工就没饭吃，账房担心库存和工资，战时负责人盯着设备清册。年轻工人问：“今天拆了，今晚拿什么开工？”工人代表答：“今天拆，是为了明天还有地方开工。”负责人把清单摊开：卷烟机、切丝机、蒸叶机、烟叶、卷烟纸、锡纸，每一项都对应着明天还能生产的可能。警报声几次打断清点，工人从慌乱转为分工：拆机、编号、装箱、找船、守夜班。

第二段为江上转运。木船、纤绳、烟箱和机器零件构成码头与江面，船工说水道险，工人说机器不能丢。多媒体提示长江上行、险滩、水道和迁运艰难。年轻工人一度脱力，问“这么慢，真的赶得上吗”；工人代表接过绳子：“慢也要走。机器走一寸，厂就活一寸。”拉纤号子与设备清单交替出现，行动紧张、台词克制，把压力一路推向重庆。

第三段为重庆厂战火中开工。场景转到重庆南岸弹子石，抢运来的设备刚刚拼起，灯刚亮，警报再响。轰炸后，厂房受损，机器残缺，工人短暂沉默。年轻工人看着残损机器，说“又毁了一次”；账房翻开清单，报出还剩的设备和原料；负责人回答：“剩多少，就用多少。今天两箱也开，明天六箱也开。机器响一声，人心就稳一分。”

这一段把前两段的疑问统一起来。拆机，是为了把生产能力从战火里抢出来；转运，是为了把这点能力送到还能开工的地方；重庆重开工，是把能力重新交还给战时中国。工人代表接上一句：“厂房能塌，机器能坏，可中国人自己造的这口气，不能断。”众人重新接电、扶正机件、整理烟叶，机器声很弱，却在众人的注视中逐渐稳定。

第三幕的情绪落点是重庆厂重开机。战时负责人、工人代表、账房和年轻工人围着机器，胜利感来自“还能开工”这个共同看见的事实。南烟的火种在这一刻升为民族工业的火种：战火压下来，实业仍要转；城市被炸过，人心仍要稳；国货的名字仍要留在中国人自己手里。

#### 七、尾声 ——【火种交出去】

重庆厂的机器声逐渐稳定，情景剧在人物对话中完成尾声。账房把搬迁清单、残留烟标和开工记录放在同一个档案夹里。年轻工人问：“这些以后还有人看吗？”负责人回答：“有人接着做，就有人看得懂。”简照南、简玉阶或其精神影像在光中短暂回望，第一幕账本里的白鹤标识与第三幕的设备清单合在一起，形成一条从失败到护火的线。

多媒体短促点到：1980年南洋烟草产权关系正式归属上海市政府，1981年上海实业有限公司在香港注册成立，南洋烟草、永发印务、香港天厨等百年民族工业企业成为重要基础。信息点到即止，情景剧的结尾仍落在南烟人物身上：工人代表把手放在重新响起的机器上，说“火还在，家国的希望就还在。”

随后进入节目衔接。歌舞演员从机器声和灯光中进入，接过档案夹、火光或机器节拍。南烟人物退到侧后方，完成第一段语言节目；后半歌舞接续祖业群像与上实45年的整体表达。舞蹈承担承接与放大，情景剧的故事已经在“重庆重开工”与“火还在”中完成。

### 多媒体与舞蹈衔接

台词承担人物选择：兄弟的动摇与再起、简铭石的现实支持、谈判桌上的内部分歧、战时工人围绕机器和搬迁的判断、重庆重开工后的家国精神落点。

多媒体承担时间、地点和复杂背景：白鹤纠纷、并购条款、股权比例、战争新闻、内迁距离、1980/1981上实承接信息。

舞台蒙太奇承担时间跨度和发展质感：国货热潮、商号来信、渠道铺开、广告烟画、街头推广、第二幕三次谈判之间的实力变化。

舞台动作承担可感的戏剧动作：巡理府前当众销毁、账本清算、机器重启、合同合上、谈判间蒙太奇、拆机装箱、逆水拉纤、轰炸后重开工、节目衔接。

情景剧以情节和台词完成自身收束。后半歌舞承接情绪和视觉能量，作为第二个节目展开祖业群像。
