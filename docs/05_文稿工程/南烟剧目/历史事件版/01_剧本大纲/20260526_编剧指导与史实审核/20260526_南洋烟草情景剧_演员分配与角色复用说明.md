# 南洋烟草情景剧演员分配与角色复用说明

更新时间：2026-05-26  
适用版本：后续v14及正式台本修订
用途：整理当前角色口径，降低非核心配角的贯穿负担，保留尾声员工整理资料情节。

详细有台词角色、演员编号、各幕出场和10人以内压缩方案，见同目录 `20260526_南洋烟草情景剧_演员表规划_v6.md`。本文件保留原则说明，新文件作为台本阶段演员表规划入口。

## 一、总体原则

本剧角色分为三类：明确历史原型角色、南烟员工群像、时代环境群像。

明确历史原型角色保持姓名和戏剧功能稳定。其他角色不必设置为贯穿配角，可按场景需要使用“员工”“工人”“商贩”“报童”“船工”“董事/股东”等身份指代。演员可在不同幕中复用，观众只需看出其所处身份和时代功能。

“账房”不再作为贯穿人物重点塑造。账本、清单、合同、档案等道具仍保留，但关键动作应由简氏兄弟、陈荣贵或员工群像共同承接。由于各幕之间时间跨度很大，员工角色不作为同一人物贯穿；演员可以分饰不同时代的员工角色，但台本中必须按具体角色身份书写，例如“初创员工”“战时老员工”“尾声青年员工”，不再写成“员工A类”“员工B类”。尾声两个员工整理资料的对话保留，作为情景剧收束和历史承接的重要情节。

## 二、固定历史原型角色

| 角色 | 出场范围 | 功能 | 演员口径 |
| --- | --- | --- | --- |
| 简照南 | 引子、第一幕、第二幕 | 初创行动者，失败承受者，商战选择者 | 固定演员，不与其他角色复用；第三幕只作旧照、烟标、账本或声音回响 |
| 简玉阶 | 引子、第一幕、第二幕、过场二、第三幕暗线 | 兄弟关系中的情绪力量和守业承诺，战时拒绝拉拢的精神暗线人物 | 固定演员，不与其他角色复用；第三幕以年老形象进入侧台暗线 |
| 简铭石 | 第一幕 | 现实支持者，帮助兄弟从失败中重新行动 | 可由独立演员承担；演员紧张时可在后段复用为非姓名群像 |
| 陈荣贵 | 过场二、第三幕 | 汉口厂厂长、后任重庆制造厂厂长，战时迁厂和复工的现实行动人物 | 建议固定演员，可由原第三幕行动核心演员承担 |

说明：简照南、简玉阶是舞台主轴。简铭石本人只作为第一幕历史人物使用；资料暂不支持他参与1914至1922年前后的并购谈判。第三幕进入抗战内迁阶段后，简照南不作为现实行动人物进入战时现场；简玉阶不与陈荣贵安排现实会面，以拒绝日寇拉拢、守住民族尊严和牵挂同胞同事形成精神暗线。战时现场的现实行动由陈荣贵和员工群像承担。

## 三、南烟员工群像

南烟员工群像承担工厂、账目、设备、档案和劳动现场，不设置过强个人传记。台本阶段应按角色身份切换称呼：

| 角色口径 | 可承担场景 | 功能 |
| --- | --- | --- |
| 初创员工 | 引子、第一幕工厂现场 | 承接初创期工厂现场和员工饭碗压力 |
| 初创青年员工 | 第一幕心理压力、清算现场 | 承接第一幕青年视角，不延续到尾声 |
| 第二幕工厂员工 | 第二幕谈判边缘、蒙太奇 | 用一句现场压力把谈判落回工厂 |
| 战时老员工 | 第三幕汉口拆机、江上转运、重庆复工 | 承接经验、责任和“机器也要走”的判断 |
| 战时青年员工 | 第三幕战时现场 | 承接撤离恐惧、生命压力和复工疑问 |
| 尾声老员工 | 尾声档案整理 | 承接历史记忆和“有人接着做”的回答 |
| 尾声青年员工 | 尾声档案整理 | 后辈提问者，不是第一幕初创青年员工 |
| 负责清点的员工 | 第一幕清算、第二幕递合同/账本、第三幕清点设备 | 提供事实信息，不成为独立主线 |
| 搬运/机修员工 | 第一幕重启机器、第三幕拆机装箱、重庆重开工 | 承担动作场面和工业质感 |

后续修订时，可把“账房/档案员”改为“负责清点的员工”“员工递上账本”“员工翻开清单”等动作表达。这样保留信息功能，也避免账房抢走简氏兄弟的戏剧主体。

## 四、时代环境群像

| 角色口径 | 主要出场 | 功能 | 复用方式 |
| --- | --- | --- | --- |
| 报童/送信人 | 过场一、过场二、第三幕，可在引子短暂出现 | 带出新闻、时代转换和多媒体启动；第三幕在日寇画外音施压后向简玉阶递急信 | 可由员工或环境演员兼任 |
| 烟贩/商号伙计/买主 | 引子、第一幕返场 | 呈现街面信任、恐惧和谨慎支持 | 可由同一组群像演员轮换 |
| 董事/股东复合角色 | 第二幕 | 呈现内部现实压力和资金焦虑 | 可由简铭石演员后段换身份复用，服装区分 |
| 外部强权画外音 | 引子、第二幕、过场二、第三幕暗线 | 呈现外商指控、资本诱惑和抗战时期拉拢威逼 | 只用画外音、字幕、合同文本、打字机声和影像压力，不作为可见角色登台 |
| 船工/搬运工 | 第三幕江上转运 | 呈现水路艰险和迁运难度 | 可由前场商贩、员工演员复用 |
| 重庆厂员工 | 第三幕、尾声 | 呈现战火复工和精神收束 | 可由南烟员工群像延续 |

环境群像不追求姓名完整性。身份清楚、行动准确、与主线关系明确即可。

## 五、推荐演员配置

### 基础配置：8至10人

1. 简照南
2. 简玉阶
3. 简铭石/董事复用
4. 初创员工/战时老员工/尾声老员工分饰
5. 初创青年员工/战时青年员工/尾声青年员工分饰
6. 负责清点的员工/报童复用
7. 烟贩/商号伙计/船工复用
8. 商号伙计/买主/公众质询信息递送动作复用
9. 陈荣贵
10. 群像演员若干，可由歌舞演员补充

### 扩展配置：12人以上

扩展时优先增加员工和群众，不优先增加有姓名配角。第三幕可增加拆机、搬运、拉纤、重庆复工人员，形成大气感。第二幕可增加工厂边缘群像，让谈判桌外始终有产业现场。

## 六、各幕角色分配

### 引子

固定角色：简照南、简玉阶  
可复用角色：烟贩、报童、外部强权画外音

重点动作由简照南、简玉阶承接：接过样烟、上前争辩、拦住争辩、把“白鹤”放回待销毁烟箱。烟贩只用退避动作形成街面压力，员工不在引子中承担可见动作或台词。

### 第一幕

固定角色：简照南、简玉阶、简铭石  
可复用角色：员工、负责清点的员工、烟贩、商号伙计、买主

清算信息由员工群像分担报出，情绪承接落在简氏兄弟身上。重新站起的动作由简照南、简玉阶、简铭石和员工群像共同完成。

### 过场一

可复用角色：报童、商号伙计、广告/国货传播群像、员工

报童与群像负责快剪，报童可作为固定的新闻/时间转场功能处理。演员可从第一幕民众群像直接转换。

### 第二幕

固定角色：简照南、简玉阶  
可复用角色：董事/股东复合角色、负责清点的员工、工厂员工群像、报童、公众质询信息、外部强权画外音

谈判核心仍在简氏兄弟。负责清点的员工只递账本、合同、名册，不承担判断。工厂员工群像可在谈判桌外的光区或蒙太奇中出现，表示产业现场。外部资本压力只用画外音、合同文本、字幕和打字机声，不设置可见外商代表。

### 过场二

可复用角色：报童、陈荣贵、员工、负责清点的员工、日寇拉拢声音

过场二以多媒体和报童战报为主，字幕交代“南烟落入他手、上海厂毁于战火、汉口厂必须西迁”，随即由陈荣贵展开设备清册进入迁厂行动。年老简玉阶的处境不在过场二堆叠，移入第三幕由日寇画外音先施压，送信人后置递入迁厂消息。

### 第三幕

可复用角色：年老简玉阶、日寇拉拢声音、送信人、陈荣贵、战时老员工、战时青年员工、负责清点的员工、搬运/机修员工、船工、重庆厂员工

第三幕不需要每个员工角色拥有固定姓名。简玉阶线作为精神暗线，用侧台、画外音、旧账本、佛灯、急信或旧烟标呈现，不参与现实迁厂指挥。日寇拉拢与威逼先由画外音完成，送信人只承担后置事实递送，可由报童或环境演员分饰。陈荣贵作为有名有姓的现实行动人物，承担迁厂、清点、复工判断。员工群像通过分工呈现战争压力：有人关心家人撤离，有人清点设备，有人拆机，有人找船，有人扶机器。年轻员工的疑问从生命和撤离出发，最终落到重庆复工。若由第一幕演员复用，应通过有胡须的战时老员工、战时青年工人、机修工等扮相变化切开人物身份，并在台本角色名上直接写成“战时老员工”“战时青年员工”等具体角色。

### 尾声

保留两个员工整理资料的对话。角色可设为“尾声老员工”和“尾声青年员工”，也可写为“档案整理老员工”和“后辈员工”。尾声青年员工不应写成第一幕中出现过的同一名青年员工；可以由同一演员承担，但角色身份属于尾声档案时空。

保留情节口径：

> 员工整理搬迁清单、残留烟标和开工记录。青年员工问：“这些以后还有人看吗？”老员工回答：“有人接着做，就有人看得懂。”

这个情节承担从历史现场转入档案传承的功能，后续修订不删除。多媒体随后点到1980年南洋烟草产权关系归属上海市政府、1981年上海实业有限公司在香港注册成立。

## 七、后续v14及台本修正方向

1. 角色表中弱化“账房/档案员”，改为南烟员工群像下的账目、清单动作。
2. 第一幕和第二幕中，账本、合同、清单的关键判断动作转给简照南、简玉阶；第三幕的现实行动判断转给陈荣贵。
3. 第三幕改为“陈荣贵现实迁厂 + 简玉阶独立困局 + 重庆复工合流”，不写简玉阶与陈荣贵现实会面。
4. 第三幕中员工不固定姓名，以战时工厂群像呈现，演员可复用，但人物身份不与第一幕强行贯穿。
5. 尾声保留两个员工整理资料的对话，并作为情景剧独立收束的一部分。
6. 扩大员工和群众数量时，优先服务第一幕民众鼓励、第二幕工厂现场、第三幕战时复工的大气感。
7. 所有外籍强权均退为声音和影像压力，台上人物面向观众回应，让观众第一视角感受历史压力。
