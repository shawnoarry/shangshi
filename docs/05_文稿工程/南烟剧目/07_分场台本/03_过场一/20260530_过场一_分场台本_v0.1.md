# 南洋烟草开场情景剧《火种》

## 过场一 —— 分场台本 v0.1

【舞台呈现形式】

本段控制在约 45 秒。由第一幕重新开机的机器声转入国货热潮，以报童、烟箱、商号来信和冰屏烟标完成时间推进，把南烟推入第二幕的商战长桌。

【角色需求】

报童、商号伙计、南烟员工群像。

【多媒体及舞美方向】

舞台保持同一装置。机器区继续运转，员工推烟箱穿过不同光区。冰屏快速出现白鹤残标、双喜、飞马、红双喜烟标、国货广告语、市场地图和订单字样。中区长桌在本段末尾推入，桌上落下烟标、地图、订单和报价纸。

【剧本正文】

【第一幕机器声延续。报童举着报纸从台口跑过。】

报  童  新报！国货展上有新声！中国人请吸中国烟！

【冰屏闪过国货展、报纸广告、白鹤残标、双喜、飞马、红双喜烟标。员工推烟箱穿过舞台，商号伙计从另一侧递来信袋。】

商号伙计  山东来信，要购南烟。

商号伙计  台州商号来信，愿推销飞艇。

商号伙计  上海也问货，只说申地还缺经售机关。

【冰屏地图点依次亮起：香港、南洋、山东、台州、上海。烟箱被推向地图亮点，上海点前出现“经售机关”字样。】

报  童  双喜上柜，订单进厂！

【机器声与打字机声交织。中区长桌被推入，订单、地图、烟标落桌。一张报价纸从高处落下，压在订单上。】

【灯光转向长桌。第二幕开始。】
