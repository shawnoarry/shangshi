## 一、会议背景

本次讨论围绕《火种》第二幕继续调整。此前版本已将第二幕从单纯外商并购谈判，逐步扩展到红双喜、国货发展、教育公益、红色基因等内容。但继续梳理资料后发现，许多能够体现“支持革命”“红色基因”的事件发生在1922年之后，甚至延伸至1925年及更后阶段，与第二幕原本围绕1914、1917、1922年三轮并购谈判的时间线存在衔接难点。

会议最终形成的基本判断是：第二幕仍应保留与外资谈判这条对外冲突线，但篇幅需要缩小。谈判承担第二幕的戏剧骨架，原先放在蒙太奇里的红双喜发展、支持教育、公益责任、社会革命思潮等内容，需要挑选更具体的事件进行舞台演绎。

## 二、第二幕调整方向

1. 保留外资谈判作为第二幕核心冲突支点。  
导演判断，谈判桌是一个有舞台张力的对外冲突场景，能够集中表现南烟面对外资强权时的民族立场。它可以成为一个“小缩影”，再由此扩展到外部社会空间，呈现国货、工人、教育、革命思潮等时代背景。

2. 压缩前两段谈判。  
1914年收买试探、1917年合并条件可以保留，但不宜展开太重。重点表达南烟面对外资收买和合并压力时的态度，以及简氏兄弟的民族工业立场。前两段可作为第二幕前半部分的推进节点。

3. 将蒙太奇内容转为更具体的事件演绎。  
原先“红双喜发展、国货传播、公益、教育、革命思潮”等内容若只用大屏快剪，会显得像资料罗列。本轮方向是选择可上舞台的动作，例如烟标、报纸、账本、商号来信、教育捐助、工人运动风潮等，让观众能看见事件本身。

4. 第二幕末端落到符合年代的社会整体革命思潮。  
支持革命部分不宜写成简氏兄弟直接领导革命运动。更稳妥的写法，是让南烟置身于20世纪20年代初上海和烟草行业的社会风潮之中，表现他们支持国货、重视教育、回应工人权益和顺应时代进步。

5. 保留兄弟二人的精神交流。  
会议提出，第二幕结尾可在全场社会情绪昂扬之后，切入简照南与简玉阶的意识空间。简照南逐渐退场，简玉阶站到兄长原来的位置上，形成精神交接。这一处理既承接第三轮谈判后简照南身体衰弱的史实，也保留《火种》原有的兄弟情感线。

## 三、关于“支持革命”部分的讨论

会议中重点讨论了几个可用史实：

1. 1921年英美烟草工人罢工。  
资料中记载，英美烟草工人罢工是中国共产党领导的早期工人运动之一，随后上海烟草工人会成立。这一事件发生在南烟第三次并购谈判之前，时间上更接近第二幕，可作为“时代革命思潮已经进入烟草行业”的背景。

2. 1922年南洋烟草职员同志会和17条会谈。  
资料中记载，1922年11月6日，南烟厂6000余人参加怠工并提出17项条件；次日下午，简照南与工人代表在总经理室会谈，接受工人要求，当日下午4时复工。该事件戏剧性很强，但会议也指出，它更像企业内部劳资/权益问题，与“支持革命”之间需要谨慎建立逻辑，不宜直接作为革命支持的主证据。

3. 五四运动相关内容。  
会议认为，五四期间更需要请领导判断的是“敬告国人”与日资、日籍身份风波的敏感性。该事件能体现南烟公开自证、公开账目、向国人招股，但其中涉及简照南曾经加入日籍、英美烟草借此攻击南烟为日资等内容，正式提报前需要明确可讲尺度。

4. 五卅运动相关内容。  
会议倾向认为五卅时间较后，且政治敏感度较高，暂不作为当前第二幕重点展开。若后续领导明确希望强化红色革命线，再另行判断是否进入尾声快剪或后续版本。

## 四、对上沟通重点

会议提出，下一步应先向领导说明当前创作取舍：

1. 为什么仍保留谈判。  
保留谈判是为了保留对外戏剧冲突，通过外资压力凸显国人自强、国货自立和“中国人做中国烟”的精神。谈判篇幅会缩小，并与红双喜、教育公益、社会风潮等事件结合，不会把第二幕写成单纯的并购史。

2. 为什么支持革命部分初步选取早期工人运动。  
1921至1922年前后的烟草行业工人运动与第二幕时间更接近，也能与上海厂、高阳大楼、南烟员工群像形成舞台联系。表达上应写成南烟身处时代风潮、顺应进步潮流，而不是把南烟写成革命运动的直接发起者。

3. 需要领导判断的敏感点。  
五四期间“敬告国人”、日资日籍身份风波、公开脱离日籍、公开账目和招股等内容，史实价值很高，但涉及身份与舆论攻击，建议请领导明确是否可写、写到什么程度、采用何种表述。

## 五、后续写作建议

建议下一版先按较稳妥口径形成第二幕剧情梳理：

1. 谈判作为骨架，前两轮压缩。
2. 红双喜、国货、教育公益从蒙太奇变为具体舞台事件。
3. 革命思潮以1921至1922年前后的烟草行业和工人运动作为时代背景。
4. 1922年第三轮谈判与兄弟精神交接作为情感收束。
5. 五四身份风波和五卅运动暂列为待领导判断内容，不在未确认前重写成强戏。

会议后另有关于其他节目方向的讨论，涉及青春活力、上实大厦建设等，暂不纳入本次《火种》第二幕整理范围。
