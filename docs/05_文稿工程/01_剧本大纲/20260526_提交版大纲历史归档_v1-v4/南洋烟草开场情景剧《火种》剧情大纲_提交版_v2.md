# 南洋烟草开场情景剧《火种》

## 剧情大纲

### 核心主旨

本剧以南洋烟草早期发展中的三次关键危机为主线，讲述简氏兄弟和一代民族实业人在失败、诱惑与战火中守住民族工业主体性的故事。作品聚焦上实45周年特别活动中的一个语言类节目段落，用南洋烟草的祖业故事为整场活动点起一束“火”：实业报国落在具体行动里，落在机器、品牌、员工、信用和希望被一次次护下来的选择里。

全剧围绕三个问题推进：败了，还办不办；卖了，能不能守住根；毁了，还能不能重新开机。三次追问层层加压，形成从经营失败、资本吞并到战时内迁的戏剧递进。舞台上，火、机器、账本和合同贯穿始终，让“国货”“实业”“祖业”成为可见、可听、可被人物承担的舞台动作。

### 剧情概述

1905年前后，年轻的简照南、简玉阶兄弟在香港旧货仓点起民族烟草工业的第一缕火。白鹤牌初有起色，却因商标纠纷被迫销毁，公司陷入清算。失败之后，兄弟在叔父简铭石和工人、账房的现实支持中重新站起，选择东山再起。

随着南洋烟草在国货热潮、渠道扩张和上海工厂建设中逐渐壮大，外资巨头多次抛出收买与合并条件。南烟内部也出现现实分歧：资金、扩厂、工资、市场风险，每一项都压在谈判桌上。简氏兄弟在诱惑与压力中守住企业主体性，让“国货”从一句口号变成必须承担的信用。

抗战时期，战火逼近，汉口厂抢救式搬迁，机器和物资沿长江西迁，重庆厂又在轰炸中重新开工。全剧以“厂房可以毁，火种不能灭”的情绪收束，把拆机、抬箱、拉纤、开机的动作交给后半歌舞，转入百年祖业与上实45年承接的昂扬表达。

### 整体风格与结构

本剧为历史情景剧，整体气质热烈、紧凑、厚重。舞台表达采用“现实场景 + 心理空间 + 舞台蒙太奇 + 多媒体信息层”的组合方式。外部压力通过画外音、字幕、合同文本、报纸标题、警报声和冰屏影像呈现，舞台正面重点呈现简氏兄弟、家族支撑、工人群像和企业内部选择。

建议时长按12至15分钟控制，并与后半歌舞组成开场组合节目。结构为“引子 + 三幕 + 两段过场 + 尾声转歌舞”：

- 引子：白鹤初燃又被迫销毁，形成第一记冲击。
- 第一幕：失败清算后的东山再起，回答“败了，还办不办”。
- 过场一：国货热潮、渠道扩张、上海升级，说明南烟为何成为对手。
- 第二幕：多轮并购谈判，回答“诱惑和压力下卖不卖”。
- 过场二：时代转暗，商战压力转为战争压力。
- 第三幕：战时内迁与重庆重开工，回答“毁了，还能不能留下”。
- 尾声：火种交给歌舞，点到上实承接，打开后半祖业群像表达。

### 出场人物

1. 简照南：史料中有明确原型的主要人物。作为全剧主行动人物，他前期有闯劲和实业理想，失败后有自责与动摇，第二幕在现实压力与民族主体性之间完成成熟选择。

2. 简玉阶：史料中有明确原型的主要人物。作为兄弟关系中的情绪力量，他更直接、更不服输，可承担第二幕中反对合并、守住根基的强烈表达。

3. 简铭石：史料中有明确原型的主要人物。作为第一幕精神支点，他提供现实支持，也把兄弟从失败情绪中拉回可执行的行动。

4. 账房/档案员：有资料基础的复合角色，不对应单一史料人物。第一幕清点债务和机器物料，第二幕把账本、员工名册或公开账目放上谈判桌，第三幕转为设备清册和搬迁清单。

5. 工人代表与工人群像：有资料基础的复合角色，可按不同年代设置为多名工人代表。第一幕代表初创工人和失业焦虑，第二幕代表员工饭碗和工厂命运，第三幕成为战时拆机、搬运、重开工的行动力量。工人角色可适当扩充人数，支撑开场节目的大气场面。

6. 董事/股东复合角色：第二幕内部冲突人物，可分主和、主战、务实三种声音，代表资金、扩厂、工资和市场风险等现实考量。

7. 环境群像：报童、烟贩、商号伙计、商会代表、学生代表、广告宣传员、街市商贩、搬运工、船工、重庆厂工人等，承担时代背景、市场流动、国货气氛、战时转运和歌舞衔接。

舞台主要角色以简氏家族为核心。资料中另有明确姓名的历史人物，可根据台本需要弱化为合同署名、报纸字幕、画外音或背景信息。环境戏和舞台蒙太奇中的配角以身份指代即可，避免舞台重心从简氏家族和南洋烟草三次危机上分散。

### 场景设置

1. 香港码头/罗素街旧货仓外场：用于引子。市井声、搬货声、报童声和洋烟广告共同构成初创环境。

2. 清算仓库/账房空间：用于第一幕。旧机器、账本、烟标、牌匾、物料箱构成主要道具。

3. 心理压力空间：用于第一幕前半段。冰屏、报纸标题、重叠人声和四面方位调度构成压迫感。

4. 国货发展蒙太奇空间：用于过场一。演员调度、地图点亮、广告画翻转、烟箱传递共同呈现企业做大。

5. 谈判桌：用于第二幕。同一张桌子串联多轮并购谈判，通过合同厚度、灯光方向、人物站位和服装细节表现时间推进。

6. 战时工厂/码头/江面/重庆厂：用于第三幕。机器架、木箱、纤绳、船板、警报声和烟尘灯光完成多空间转换。

7. 档案光/歌舞入口：用于尾声。搬迁清单变成档案盒，机器声变成音乐节拍，火光变成城市光。

### 剧情大纲

#### 一、引子 ——【白鹤初燃】

舞台从香港码头或罗素街旧货仓外的热闹场景开始。汽笛、叫卖、搬货、报童声交织，年轻的简氏兄弟穿行其间。新做好的“白鹤”烟标被工人举起，烟贩说街面上有人开始问这个牌子。简玉阶兴奋，简照南望向远处洋烟广告，意识到真正的对手还在前方。

兄弟短对话点出初心：洋烟一船一船进来，中国人的钱一船一船出去，中国人自己造、自己卖、自己被国人信任，这件事有了超出生意的分量。舞台先让观众看见朝气和希望。随后氛围突然转冷，外商指控以画外音进入，冰屏出现法庭、商标、败诉、销毁等纪录片式字幕。巡理府前，工人本能护住烟箱，烟贩怕被牵连，简玉阶要上前理论；简照南拦住他，把烟贩的欠账记进账本，又从工人手中接过一包“白鹤”，放进被勒令销毁的箱内。他站在烟箱前承下这一次羞辱。刚才那句“让国人信任”的理想，在这一刻变成他对工人、烟贩和街面买主的欠账与责任。火光以写意方式映出白鹤标识的崩塌，字幕明确为“价值2000多元‘白鹤’牌香烟在港英当局巡理府前当众销毁”。

引子结束时，火色还在，简照南把残留的一张白鹤标识夹进账本，简玉阶看着空了的烟箱。灰烬中的微光接入第一幕。

#### 二、第一幕 ——【败后再起】

第一幕直接承接引子末尾的火色和账本。简照南把残留的白鹤标识夹进账本后，舞台不立刻进入现实清算，而是先进入抽象心理空间：四面声音压上来，外商声音、报纸标题、烟贩低语、工人追问、街面议论彼此叠加。冰屏上反复出现“败诉”“仿冒”“销毁”“清算”“谁还敢买”“谁还敢卖”等词。简玉阶想冲出去辩白，却被一圈圈人影拦回；简照南抱着账本站在中央，所有声音都像在向他追债。

这一段要把兄弟的挫败感具象化。简玉阶的不甘可以更外放，他不断追问“凭什么”；简照南的压力更内收，他听见的是工人明天的饭碗、烟贩不敢摆货的眼神、街面买主被牵连后的沉默。前面“让国人信任”的理想，在心理空间里反过来成为最重的压力：信任一旦被打掉，受伤的是一个牌号，也是所有愿意试一试国货的人。

心理压力达到顶点后，现实空间才逐渐回到清算日。账房报出机器、物料、债务和无人敢买的现实，工人不知明日去处。简铭石上场，把他们从心理空间拉回仓库现实。他先问机器在哪里、物料在哪里、债怎么还、人还剩几个，再以现实支持和精神提醒推动兄弟重新站起。简铭石的帮助既是资金和资源上的托底，也是信念上的托举：败一次不可怕，怕的是中国人自己认定中国人办不成。

后半段加入民众返场，让东山再起的决心同时来自家族、员工和街面上对国货的期待。引子中的烟贩、商号伙计、报童和买主可以从心理空间的质疑声中转化出来：先是犹豫地把空烟盒、赊账条或旧订单放回桌上，再有人说“等你们再开张，我还摆出来”，也有人说“这次不敢摆在柜面，下次你们把牌子做稳，我第一个问货”。这些声音不必写得整齐高亢，要带着市井里的谨慎、同情和不服气，与前半段“谁还敢买、谁还敢卖”的质疑形成对比。

这一幕要从“精神崩塌”写到“重新开机”。前半是心理压力具象化，后半是现实空间回归和群众情绪托举。账房在账本上划掉旧债、写下新章程，简铭石把现实支撑放上桌，工人掀开蒙住机器的布，烟贩和商号伙计把旧订单重新压在账本旁。几只手把钱、章程、印章、工具和订单放到同一张桌上。旧机器重新启动，机器声从断续到稳定，第一束暖光亮起。

#### 三、过场一 ——【火怎样烧起来】

机器声不断，转为国货热潮的节拍。报童举起国货新闻，商号递来订单，烟箱在演员手中传递，地图上点亮南洋、广州、上海、南京、镇江、苏州等节点。演员推转长桌，旧仓空间变成销售桌、广告桌和工厂桌，高阳大楼与上海制造厂剪影出现。

过场一完成从“重新站起”到“被外资盯上”的逻辑。观众需要看见，南烟靠产品、渠道、广告、上海工厂和公众信任做出了声势。越来越亮的地图和广告被合同条款、股权比例和报价数字覆盖。外商画外音从“指控”变为“合作”，长桌被推到舞台中央，进入第二幕谈判。

#### 四、第二幕 ——【和战之间】

同一张谈判桌串起三轮并购谈判。外商以画外音、打字机声、合同文本和报价数字施压，舞台上的正面冲突发生在南烟内部：兄弟、董事、账房和工人代表围绕企业主体性展开争论。

第一轮，外商抛出收买条件，带着威胁也带着诱惑。南烟刚刚站稳，简照南以高价回应，既拖住对方，也显示这门买卖没有廉价退路。这一轮强调弱势中的机智。

第二轮，外方提出更完整的合并条件。内部出现分歧：联合可以换来资金和安全，守住主体性意味着继续面对巨头压力和未知风险。简照南必须面对扩厂、工资、债务和市场，简玉阶承担更直接的情绪表达。身份危机等敏感背景如需保留，可压缩为报童传报、公开账目、招股文件、印章改组等动作组或短字幕，台词重点仍放在谈判桌上。

第三轮，合同更厚，条件更诱人。外商声音提出资产合并、品牌保留、仍由简氏经营等看似周全的方案。工人代表把员工名册、机器清单和账本放到桌上：如果留下品牌名字，却交出制造权和未来，南烟的根就被抽走了。报价纸被账本压住，合同合上，没有签。机器声重新盖过打字机声。

第二幕的落点在现实压力中的承担：南烟可以缺钱，根不能丢；国货可以艰难地活，主体性不能交出去。

#### 五、过场二 ——【时代压下来】

谈判桌上的合同纸变成报纸。屏幕上的商业新闻、广告和招股信息逐渐被九一八、淞沪、卢沟桥、民族工业内迁等战争信息取代。舞台色调由暖转冷，警报声进入。

这一段负责从商战转到战时。第二幕守住了主体性，第三幕要守住还能生产的能力。演员把谈判桌拆成木箱、机器架和船板，账本变成搬迁清单。刚刚压住合同的账本，此刻被打开为设备清册。一声警报后，战时负责人或工人代表喊出“先拆能走的机器”，第三幕从拆机动作开始。

#### 六、第三幕 ——【江上护火】

汉口厂内，一面还要维持生产，一面开始拆卸设备。工人问机器这么重、船这么少，真的要搬吗。账房报出卷烟机、切丝机、蒸叶机、烟叶、卷烟纸、锡纸等清单。负责人回答：搬走的是明天还能转起来的可能。

第一段为拆机。演员分组拆机、清点、包装、找船、维持夜班生产。机器声和警报声互相抢节奏，时间、重量、船只、警报和人的体力共同构成压力。

第二段为江上转运。舞台转入江面和码头，木船、纤绳、烟箱和机器零件以抽象道具呈现，演员身体形成逆水拉纤的线条。多媒体以字幕或影像提示长江上行、险滩、水道和迁运艰难，让观众感到“火种”很重、很慢、很险，是被人用身体护下来的。

第三段为重庆重开工。场景转到重庆南岸弹子石。刚运来的设备重新开动，灯刚亮，警报再响。轰炸后，厂房受损，机器残缺，工人短暂沉默。沉默之后，有人扶起机器零件，有人转移烟叶，有人重新接电。靠仅存设备，重庆厂再次开工。机器声很弱，但终于出现。

第三幕的情绪落点是重庆厂重开机。机器声先弱后稳，拉纤号子、工人呼吸和机器节拍融合成音乐节奏。

#### 七、尾声 ——【火种交出去】

重庆厂的机器声逐渐稳定，拉纤号子、工人呼吸和机器节拍融合成音乐节奏。舞台出现一次时间折叠：搬迁清单变成档案盒，战时账本变成后来接续祖业的文件。火光从战时灯火转为档案室白光、香港窗口灯光和城市光。

多媒体或画外音简洁点到：1980年南洋烟草产权关系正式归属上海市政府，1981年上海实业有限公司在香港注册成立，南洋烟草、永发印务、香港天厨等百年民族工业企业成为重要基础。此处只作点睛，情绪交给后半歌舞。

歌舞演员进入，接过火光、账本或机器节拍。南烟人物退到侧后方或融入群体，个人与企业故事进入更大的祖业群像。舞蹈从拆机、抬箱、拉纤、开机的动作中长出来，转向更明亮的香港城市光、上海窗口和百年祖业焕新的气象。

### 多媒体与舞蹈衔接

台词重点承担人物选择：兄弟的动摇与再起、简铭石的现实支持、谈判桌上的内部分歧、战时工人围绕机器和搬迁的判断。

多媒体重点承担时间、地点和复杂背景：白鹤纠纷、并购条款、股权比例、战争新闻、内迁距离、1980/1981上实承接信息。

舞台蒙太奇重点承担时间跨度和发展质感：国货热潮、商号来信、渠道铺开、广告烟画、街头推广、谈判之间的实力变化。

舞台动作重点承担可感的戏剧动作：巡理府前当众销毁、账本清算、机器重启、合同合上、拆机装箱、逆水拉纤、轰炸后重开工、动作转歌舞。

### 后续深化提示

进入正式台本前，年份、金额、人名、机构名、引文和舞台字幕需回看原始资料逐条精校。尤其是白鹤商标纠纷、简铭石帮助复办、多轮并购谈判、汉口厂内迁、重庆厂重开工、1980/1981上实承接等信息，应以画册原页和整理索引为准。
