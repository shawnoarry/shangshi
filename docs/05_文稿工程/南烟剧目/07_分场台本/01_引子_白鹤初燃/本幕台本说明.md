# 引子 ——【白鹤初燃】

## 基准内容

以 2.0 提交审核版为准。舞台从香港码头或罗素街旧货仓外的热闹场景开始，白鹤烟标被街面初识，随后商标诉讼和当众销毁形成第一记冲击。

## 台本功能

- 建立简氏兄弟、香港旧货仓、洋烟压境和民族工业初燃的环境。
- 用“白鹤被烧”让观众迅速理解南烟创业的第一重压力。
- 为第一幕的心理坠落和东山再起留下情绪入口。
