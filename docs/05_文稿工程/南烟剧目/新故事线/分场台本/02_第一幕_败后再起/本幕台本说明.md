# 第一幕 ——【败后再起】

## 基准内容

以 2.0 提交审核版为准。商标案后，清算、债务、员工去留和街面疑惧同时压来。简铭石带来现实支撑，烟贩、商号伙计和买主返场，兄弟二人决定重新开机。

## 台本功能

- 展示失败后的心理压力和现实压力。
- 保留简铭石的支点作用。
- 用民众谨慎支持和员工围上机器的动作，让“再起”来自家族、员工和街面的共同托举。
