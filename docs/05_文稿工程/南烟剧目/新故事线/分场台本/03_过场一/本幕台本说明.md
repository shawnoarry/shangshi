# 过场一

## 基准内容

以 0530 新提交版为准。机器声转入国货热潮，报童穿场，屏幕快闪白鹤残标、“双喜”“飞马”“红双喜”烟标、国货广告语和市场地图。长桌被推到舞台中央，南烟进入更大风浪。

## 台本功能

- 完成从第一幕“重新开机”到第二幕“根脉不折”的短桥。
- 以多媒体和群像动作提示双喜问世、市场扩张和外部强权重新盯上南烟。
- 控制信息量，避免提前展开第二幕正戏。
- 不在本过场提前放入1916年孙中山“为国家挽回利权者也”的背景音，该句移至第二幕第一轮与第二轮之间的发展段处理。
