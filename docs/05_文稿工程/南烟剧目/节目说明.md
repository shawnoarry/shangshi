# 南烟剧目节目说明

> 适用节目：南洋烟草情景剧《火种》及其衍生版本
> 更新日期：2026-05-28
> 当前审核基准：`新故事线/提交审核版/0528南洋烟草开场情景剧《火种》剧情大纲2.0.docx`

## 一、目录定位

`南烟剧目/` 是南洋烟草相关情景剧的完整项目归档区。南烟相关会议纪要、执行稿、资料索引、方案储备、新增资料、灵感归档、历史版本、提交审核版和过程稿，均应归入本目录。

## 二、当前目录分工

- `00_会议纪要/`：导演会原始记录、整理纪要、待整理转写。
- `01_执行稿/`：早期执行稿、专项规划、剧情大纲指导稿。
- `02_资料索引/`：《上实百廿》时间轴、情节索引、素材池、分段概述、上实发展脉络梳理。
- `03_方案储备/`：节目母题、结构切入和可替换创意。
- `04_新增资料/`：新增网页资料、待整理资料和资料卡。
- `05_灵感归档/`：早期灵感稿、弃用方案和降权结构。
- `06_剧本大纲/`：跨版本通用的总纲、统筹大纲和综合结构稿。
- `07_分场台本/`：跨版本通用的工作台本、统筹台本和综合正文稿。
- `08_正式稿件/`：南烟项目层面的正式稿、整合稿和排练稿。
- `09_提交审核版/`：南烟项目层面的统一提交口径和对上审核稿。
- `10_版本归档/`：南烟项目层面的旧总纲、旧正式稿、旧提交版。
- `历史事件版/`：南烟《火种》历史事件版完整链路，保留旧提交口径和展开编剧基础。
- `新故事线/`：当前审核基准线，重点围绕新第二幕、红双喜、国货与时代觉醒口径。
- `三幕谈判版/`：从第二幕拆出的独立谈判剧备用方向。

## 三、使用顺序

1. 当前对上审核，先看 `新故事线/提交审核版/0528南洋烟草开场情景剧《火种》剧情大纲2.0.docx`。
2. 需要追溯导演会口径，看 `00_会议纪要/`。
3. 需要查史实、情节点、页码线索，看 `02_资料索引/`，必要时回到 `work/` 和原 PDF。
4. 需要回看早期结构和执行思路，看 `01_执行稿/`、`03_方案储备/`、`05_灵感归档/`。
5. 需要回看历史事件版完整结构、演员规划和编剧指导，看 `历史事件版/`。
6. 需要查跨版本统筹稿、总纲、整合稿或统一提交口径，看根目录 `06_剧本大纲/` 至 `10_版本归档/`。

## 四、归档提醒

南烟内部新增流程、索引、核验或提交文件时，必须先判断是“版本分支内文件”还是“南烟统筹层文件”。

1. 明确只属于 `新故事线`、`历史事件版`、`三幕谈判版` 的文件，继续放入对应分支。
2. 涉及多个分支共同使用的总纲、统筹台本、整合正式稿和统一提交稿，放入根目录 `06_剧本大纲/` 至 `10_版本归档/`。
3. 文件开头或文件名中应尽量写明适用范围：新故事线、历史事件版、三幕谈判版，或“南烟统筹层”。
