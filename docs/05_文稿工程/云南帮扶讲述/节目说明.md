# 云南帮扶讲述节目说明

> 适用节目：云南帮扶讲述相关节目
> 更新日期：2026-05-28
> 当前状态：模板库已建立，待后续文件归位。

## 一、目录定位

本目录用于承接云南帮扶讲述方向的完整文稿链路。后续采访、人物素材、会议纪要、讲述大纲、分场台本、正式稿件和提交审核版，均在本目录内独立归档。

## 二、当前目录结构

- `00_会议纪要/`：导演会、客户会、内部沟通纪要及反馈摘录。
- `01_执行稿/`：执行口径、任务拆解、工作底稿。
- `01_执行稿/00_原始稿件/`：外部直接提供的采访底稿、旧稿、扫描件。
- `02_资料索引/`：人物、地点、时间线、帮扶事实和核验说明。
- `03_方案储备/`：备选结构、叙事角度、主题切口和替代方案。
- `04_新增资料/`：后续新增采访、网页、录音转写、待整理资料。
- `05_灵感归档/`：片段灵感、弃用方案、过程废稿。
- `06_剧本大纲/`：讲述结构、剧情梗概、分幕版和展开版。
- `07_分场台本/`：分场正文、讲述稿、舞台提示和工作台本。
- `08_正式稿件/`：进入排练或内部确认阶段的正式文本。
- `09_提交审核版/`：当前对上审核、汇报和送审文件。
- `10_版本归档/`：被替代版本和历史留档。

## 三、归位规则

1. 新收到的采访底稿、原始文案、旧稿和扫描件，先进入 `01_执行稿/00_原始稿件/`。
2. 节目需求拆解、内部执行说明和工作底稿，进入 `01_执行稿/`。
3. 人物资料、帮扶事实、引用核验和时间线，进入 `02_资料索引/` 或 `04_新增资料/`。
4. 结构阶段材料放入 `06_剧本大纲/`，正文阶段材料转入 `07_分场台本/`。
5. 当前有效提交稿只保留在 `09_提交审核版/`；被替代版本移入 `10_版本归档/`。

## 四、命名建议

统一建议：

`YYYYMMDD_云南帮扶讲述_文件类型_主题或版本.ext`

示例：

- `20260528_云南帮扶讲述_人物讲述大纲_v0.1.md`
- `20260528_云南帮扶讲述_采访纪要.md`
- `20260528_云南帮扶讲述_提交审核版_v1.docx`
