# 《火种》第二幕剧情梳理 v0.2

整理时间：2026-05-27  
依据文件：`导演会/20260527215320-转写_佳。的快速会议-逐字稿文本-1.docx`、历史事件版v21、新故事线资料归档  
版本定位：第二幕结构与剧情逻辑梳理，供后续大纲修订和台本展开使用。

## 一、总体定位

原题眼“和战之间”更适合三轮并购谈判版。按照当前调整方向，第二幕的重点已经从“谈不谈、卖不卖”扩大为“一枚双喜烟标怎样带出南烟发展、国货信任、社会责任和时代觉醒”。建议第二幕暂定题眼为：

**双喜入潮**

“双喜”是这一幕的叙事入口。它既是白鹤受挫后重新拿出的牌子，也是南烟从柜台、渠道、上海厂、社会公益一路进入时代风潮的可视化物件。谈判仍保留，但不再作为第二幕的唯一主线；谈判应与双喜发展互相咬合，成为外资强权不断试图收走南烟主体性的压力来源。

这一幕要回答的问题是：当双喜、红双喜和南烟逐渐被国人接受，外部强权、公众舆论、教育公益、工人运动和社会变革同时压到这家企业面前，简氏兄弟怎样守住这门“中国人自己的烟”。

## 二、核心调整

1. 先写“双喜”发展，再进入第一次谈判。  
第二幕不宜一上来就把谈判桌推到中央。应先让观众看到白鹤之后“双喜”怎样重新进入街面：烟贩试卖、柜台下递烟、商号问货、国货热潮出现。第一次谈判由此进入才有逻辑：南烟刚刚站稳，外资才要用收买方式把它按下去。

2. 双喜发展事件与前两次谈判相交融。  
1914年第一次谈判后，南烟继续在国货展、商号来信、上海渠道、包装调整、上海分公司和上海投厂中往前走。1917年第二次谈判出现时，观众已经能看见：对方想合并的是一个正在变成国货力量的品牌、渠道和工厂。

3. 第二次谈判后逐步抬高情绪。  
1919年前后的五四运动思潮、国货身份遭质疑、公开自证、教育公益和社会义举，应作为第二轮谈判后的情绪抬升段。这里要让南烟从“商业上站住”推进到“接受国人审看、回到社会责任、顺应时代变化”。

4. 工人运动萌芽推动第三轮的情绪最高点。  
1921年英美烟草工人罢工、1922年初南烟部分高级职员响应时代风潮发起组织上海烟草工会等线索，可作为第三轮谈判前后的社会声浪。需要注意，工人运动不是遮佛士来华谈判的直接史实原因；舞台上更稳妥的处理，是让工人运动、国货思潮、教育公益和公众自证共同形成时代压力，把第三轮谈判推向态度与情绪的最高处。

5. 最后完成兄弟精神空间交接。  
第三轮谈判结束后，舞台从社会声浪收束到兄弟二人的意识空间。简照南逐渐退出，简玉阶站到兄长原来的位置上，完成守业精神的交接。

## 三、建议结构

### 第一段：双喜重新亮出来

时间：1906至1914年前后，重点压缩呈现。  
核心事件：白鹤受挫后，南烟推出“双喜”“飞马”等品牌；1909年“红双喜”诞生；辛亥革命前后国货热潮兴起，双喜等品牌逐渐打开销路。

舞台处理：

1. 白鹤残标压在旧账本里，简氏兄弟重新拿出“双喜”烟标。
2. 烟贩起初不敢摆上柜台，只从柜台下递出样烟。
3. 消费者问有没有中国人自己的烟，商号伙计带来问货消息。
4. 报童喊出国货消息，烟标、订单、铜钱、柜台在群像中快速流动。

这一段不讲全史，只抓一个动作：双喜从被压住，到重新被国人看见。它为第一次谈判提供前提。

### 第二段：第一次谈判压来

时间：1914年。  
核心事件：英美烟草提出100万港元收买南烟，并以“另有手段”施压。

舞台处理：

双喜刚刚在柜台和订单中亮起来，外部报价纸随即落到桌上。长桌被推到中央，外部强权仍用画外音、报价纸和打字机声呈现。简照南站在桌边，不急于争辩，只以资产和商誉共值300万港元周旋；简玉阶听到“收买”和威胁几乎起身，简照南按住他的手。

戏剧功能：

第一次谈判要处理成“双喜重新亮出来”之后遭遇的第一次外部压制。南烟此时仍弱，只能先稳住局面，把未来留在自己手里。

### 第三段：双喜怎样走向上海

时间：1915至1917年前后。  
核心事件：国货展、商号来信、上海无经售机关、渠道打开、包装调整、上海分公司和上海投厂。

舞台处理：

1. 国货展奖凭出现，报童和商号伙计把山东、上海、浙江台州等地来信送到台前。
2. 上海来信指出南烟产品合销路，却因上海没有经售机关而无法采购。这个矛盾要演清：有人愿买，南烟还没让他们买得到。
3. 简氏兄弟和员工开始算包装、价格、佣金、运输和渠道。10支装硬纸匣、商号利润、上海市场习惯等信息通过烟盒、价目表和账本快速呈现。
4. 地图点亮南京、镇江、苏州等地，烟箱在不同光区传递。
5. 高阳大楼和上海制造厂剪影出现，机器声逐渐压过外烟广告声。

戏剧功能：

这一段把原来偏蒙太奇的信息变成可演事件。观众要看到双喜的发展由来信、柜台、渠道、包装、工厂共同推动，而不是只听见一句“国货热潮”。

### 第四段：第二次谈判嵌入发展高点

时间：1917年。  
核心事件：较完整的合并方案出现。

舞台处理：

第二轮谈判不宜和前一段割裂。可以让一侧舞台仍在传递烟箱、订单和上海厂机器声，中央长桌则摊开合并方案。画外音和合同文本只保留几个关键点：作价、股份、仍由简氏经营、竞业限制、秘密合并。桌边人物产生分歧：有人看见资金和扩厂机会，有人担心牌号、市场和经营权被逐步收走。

简玉阶在这轮可以有更明显的情绪爆发。简照南则把账本压在合同旁，提醒众人想清楚：如果国货只剩一个壳，如果机器和牌子都听别人的，南烟还算谁的南烟。

戏剧功能：

第二次谈判要和双喜发展形成互文。南烟越被市场需要，对方越想用合并方式进入；双喜越站上柜台，谈判桌上的“归谁”就越沉重。

### 第五段：五四思潮、公众自证与社会责任

时间：1919至1921年前后。  
核心事件：五四运动思潮、国货身份遭质疑、公开自证、教育公益和社会义举。

舞台处理：

这一段从第二轮谈判后的僵持中抬高情绪。报纸、学生、商会、员工和消费者进入不同光区，社会声音开始变大。南烟面对的不只是外资谈判，也包括国人对一家民族企业的审看。

可分三组动作呈现：

1. 公众审看。  
报纸质询、账本摊开、股东名册、公开招股、公司牌匾变化依次出现。五四期间涉及日资日籍的具体表述需等待领导判断，正式大纲可先用“国货身份遭质疑”“南烟公开自证”承接。

2. 教育公益。  
一张留学船票、一方复旦简公堂铭石、一盏义学或夜学灯进入舞台。简氏兄弟把做企业所得的一部分交还教育，让“实业救国”从工厂和柜台延伸到培养人。

3. 社会义举。  
烟箱旁出现米袋、捐款戳记和赈灾账本。每售一箱捐洋五元、赈灾、平抑米价等内容不需要铺满，只需形成“卖出去的是烟，运出去的是米，留下来的是信任”的舞台印象。

戏剧功能：

这一段让第二幕从商战进入时代。南烟的双喜、红双喜不只被购买，也被国人审看；南烟的账本不只算利润，也算教育、赈灾和社会责任。

### 第六段：工人运动萌芽与第三轮谈判

时间：1921至1922年初。  
核心事件：烟草行业早期工人运动、上海烟草工人会、南烟内部组织意识萌芽；1922年初遮佛士来华推动第三轮谈判。

舞台处理：

这一段要把社会声浪推到最高。1921年英美烟草工人罢工、上海烟草工人会成立等信息，可由报童和工人群像带入。1922年初，南烟部分高级职员响应时代风潮发起组织上海烟草工会的线索，可作为上海厂内部也被时代推动的提示。舞台上，工人、学生、报童、商号、消费者和南烟员工逐渐站到不同光区，形成一个正在觉醒和变动的中国社会。

与此同时，第三轮谈判回到长桌。遮佛士带来的新方案更成熟，也更危险：双方产品统一由新公司经营，制造厂成为代制关系，按成本加佣金，新品牌共同所有，烟叶采购等环节避免竞争，资产比例看似优厚，南烟仍由简氏兄弟管理。

桌边的分歧达到最高点。有人担心资金、市场和扩张压力，有人拿出销售盈利和公众认购证明南烟还能继续打。外部画外音不断强调“合作”“优厚条件”“仍由简氏管理”，社会光区里则响起国货、教育、工人和公众的声音。

简照南病体已显，但精神最坚决。他不再陷入价格争辩，只追问：牌子归谁，机器听谁，国人认购的到底是谁的南烟。简玉阶扶住兄长，压住怒气，也接住兄长的判断。

史实边界：

1. 1922年初第三轮并购谈判与1921至1922年前后的工人运动可在舞台上并置为时代声浪。
2. 不宜写成工人运动直接导致遮佛士来华谈判。
3. 1922年11月南烟职员同志会17条会谈发生在第三轮并购谈判之后，如使用，只适合做尾部预示或资料储备，不宜放进第三轮谈判前作为因果。

戏剧功能：

第三轮谈判成为第二幕态度和情绪的最高处。南烟面对的是一份合同背后的时代追问：外资要控制，公众在审看，工人在觉醒，教育和义举已经把企业与社会连在一起。此时拒绝合并，才有足够的时代重量。

### 第七段：兄弟精神空间交接

时间：1922年初第三轮谈判后。  
核心事件：简照南将守业精神交给简玉阶。

舞台处理：

第三轮谈判声浪达到最高后，外部声音突然压低。社会光区渐暗，只留下简照南与简玉阶。简照南慢慢退出谈判桌中心，把旧账本、白鹤残标、双喜/红双喜烟标或合上的合同交到简玉阶手中。这个动作包含白鹤失败、双喜再起、外资谈判、公众自证、教育义举和时代风潮。

简玉阶站到哥哥原来的位置上，不做长篇宣讲，只给出一句承诺：厂可以再难，根不能交。随后社会光区重新亮起，工人、学生、烟贩、员工和消费者的身影与机器声一起回到舞台。第二幕在“南烟的根仍在中国人手里”的情绪中收束。

戏剧功能：

这一段让第二幕从时代大潮回到人物关系。兄弟精神交接需要回应前面所有事件：双喜为什么要守，谈判为什么不能退，教育和义举为什么有意义，时代觉醒为什么要落到简玉阶继续守下去。

## 五、关键信息取舍

保留：

1. 双喜/红双喜作为第二幕入口和视觉主物件。
2. 第一次、第二次、第三次外资谈判，作为对外冲突骨架。
3. 上海来信、国货展、渠道、包装、上海投厂等具体事件。
4. 五四思潮下的公开自证，但具体敏感词等待领导判断。
5. 教育公益和社会义举，作为民族资本家责任。
6. 早期工人运动作为时代觉醒和红色基因入口。
7. 简照南与简玉阶的精神空间交接。

压缩：

1. 第一、第二轮谈判条款。
2. 红双喜品牌史细节。
3. 公益、教育项目清单。
4. 1922年11月17条会谈及1925年五卅运动等后续内容。

待确认：

1. 五四期间“敬告国人”是否可以明确写日资、日籍、脱离日籍。
2. 支持革命是否可以写孙中山、广州国民政府、早期工人运动等具体对象。
3. 1922年11月南烟职员同志会17条会谈是否进入尾部预示。
4. 五卅运动是否只保留为后续资料，不进入第二幕正戏。

## 六、建议一句话梗概

第二幕以“双喜”发展为入口，让观众先看见南烟如何从柜台、来信、渠道和上海厂中站起来；再把第一次、第二次外资谈判嵌入南烟壮大的过程；第二轮之后，五四思潮、公开自证、教育义举和早期工人运动逐步把情绪推向整个中国社会的觉醒；最终在1922年第三轮谈判中达到最高点，由简照南把守住民族工业之根的精神交给简玉阶。
