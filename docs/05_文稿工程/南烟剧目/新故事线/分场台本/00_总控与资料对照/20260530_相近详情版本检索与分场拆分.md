## 三、分场台本展开顺序建议

1. 先写 `01_引子_白鹤初燃` 与 `02_第一幕_败后再起`，这两部分稳定度最高。
2. 再写 `03_过场一`，承担从重新开机到南烟被推入更大风浪的短桥。
3. `04_第二幕_根脉不折` 等待反馈后再进入正式台词，可先做资料卡和动作分解。
4. `05_过场二` 与 `06_第三幕_江上护火` 可先按 2.0 做分场草台本。
5. `07_尾声_薪火相传` 保留两名员工整理资料对话，并处理公私合营、1980/1981 上实承接与后半歌舞的衔接。

## 四、当前注意事项

1. 第二幕相关的“国货身份遭质疑”“敬告国人”“工人代表”“学生代表”等表达，需等待后续反馈确认尺度。
2. 过场一、过场二的功能以 2.0 为准，过程稿中的旧过场小标题只作参考。
3. 尾声 2.0 中有公私合营、员工整理档案、上实承接信息，台本展开时需处理信息重复，让舞台动作更顺。
4. 所有台词、字幕和精确史实进入正式台本前，应回看资料索引和《上实百廿》原文页码。
