# 南洋烟草情景剧编剧指导

更新时间：2026-05-26  
版本说明：v4 根据 v14至v21 口径修订，明确第二幕“三次谈判同一张桌、谈判间蒙太奇打开舞台空间”，并规定所有外籍强权不作为可见角色登台，统一由画外音、字幕、合同文本、打字机声和影像压力呈现。v20 已压缩过场二，并将简玉阶战时处境移入第三幕；v21 补强第二幕第三轮谈判中的董事和股东分歧、公众责任、简照南守护企业主体的坚持，以及简玉阶在谈判现场承接兄长托付的情感线。  
适用范围：45周年大活动中“南洋烟草”语言类情景剧段落

## 一、项目定位

南洋烟草情景剧是45周年整体演出中的一个语言类节目段落，资料整理和大纲写作均服务于这一段落。它承担“以南洋烟草历史讲述上海实业精神源流”的功能，不能替代整台45周年活动的全部语言节目策划。

本节目时长按15分钟左右控制，核心表达集中在南洋烟草从民族实业困境中站起、在商战中守住自主、在战火中保存工业火种的精神线索。全剧从引子到尾声都应聚焦中国人自强自立、不畏强权霸权的民族精神。放宽后的时长主要用于加厚人物处境、舞台蒙太奇和战时行动过程，不扩大为南洋烟草全史展示。

## 二、版本与资料管理规则

1. 每次较大修改均新起版本，不直接覆盖既有文稿。
2. 提交版大纲、内部细化大纲、史实审核、编剧指导分开管理。
3. OCR文本可用于检索和初筛，正式引用、具体页码、关键史实需回看原PDF或可靠出处。
4. 当前资料库需保持可复用状态，后续如果替换情节，不应依赖单一大纲倒推资料。

## 三、文风基础规则

1. 提交版大纲使用事实陈述和剧情呈现语言，减少内部指导口吻。
2. 避免“不是……而是……”“不……而……”等对照式句型。
3. 把“导演组如何理解”改为“观众在舞台上看到什么、听到什么、感受到什么”。
4. 避免把史实判断和创作解释直接混入提交稿，史实风险写入审核文档。
5. 历史人物台词不得写成未经出处支撑的“原话”。可写为戏剧化表达，并在审核文档中标注。

## 四、整体结构原则

当前方向以“三个关键历史节点+舞台蒙太奇+多媒体快剪”的结构推进。15分钟左右体量下，三幕结构保持不变，新增空间主要服务人物选择和舞台行动。

1. 第一部分：东山再起  
   从白鹤牌受挫、清算压力、民众情绪承接到简氏兄弟重整国货信心。第一幕的戏剧重点是失败后的主体性和再起决心。

2. 第二部分：商战谈判  
   三次谈判固定为同一张谈判桌，形成清晰的戏剧锚点。谈判之间的蒙太奇不围绕桌子打转，应打开台口、侧区、后区和高区光影，配合大屏或冰屏呈现时间流逝、企业发展和民族工业成长。谈判桌负责冲突落点，蒙太奇负责时间、空间和资料信息。

3. 第三部分：战时内迁与开工  
   以陈荣贵现实迁厂、简玉阶独立困局、重庆复工合流三段推进。第一段明确迁厂主体和行动依据，第二段讲清简玉阶腹背受敌的精神处境，第三段以重庆厂重开机作为精神统一和情感高潮。

4. 建议时长分配  
   引子约1.5分钟，第一幕约3分钟，过场一约1分钟，第二幕约4分钟，过场二约1分钟，第三幕约3.5至4分钟，尾声及转接约1分钟。若排练中需要压缩，优先压缩资讯密度和字幕停留时间，保留三次人物选择和尾声员工对话。

## 五、外部强权呈现规则

1. 外籍强权不作为可见角色登台。引子中的外商指控、第二幕中的收买与合并压力、第三幕中的日伪拉拢和威逼，统一由画外音、字幕、合同文本、打字机声、警报声和大屏影像呈现。
2. 台上人物面对这些声音时，应面向观众或向观众席方向回应，让观众以第一视角感受历史压力。
3. 不写外商代表、日寇代表与简氏兄弟或简玉阶形成写实对手戏。可写“画外音”“合同文本”“大屏字幕”“打字机声压入”。
4. 同一名配音演员可通过声线、音色和处理方式区分殖民地司法压力、外资资本压力和战时拉拢压力；该配音不计入台上演员人数。
5. 强权压力的功能是压迫中国人作选择，戏剧重心始终在简氏兄弟、陈荣贵、南烟员工和普通国人身上。

## 六、第二幕创作要求

第二幕的三轮谈判分别对应1914年收买试探、1917年汤姆士推动的较完整合并方案、1918年资产合并草案延续至1922年初遮佛士来华磋商。第三轮仍在简照南去世前，台本应呈现简照南身体逐轮衰弱、精神逐轮坚韧的变化，并将沪港董事和股东分歧、公开招股后的公众责任、简玉阶在谈判现场承接哥哥托付，合并为第二幕情感高潮。

谈判桌处理：

1. 三轮谈判使用同一张桌，建立“同一张桌上反复被逼问”的舞台记忆。
2. 每轮通过合同厚度、桌面文件、椅位、灯光方向、人物站位、服饰年代感和简照南身体状态表现时间变化。
3. 外部资本压力用画外音、合同条款和打字机声呈现，不出现可见外籍谈判角色。
4. 董事/股东复合角色负责内部现实压力，负责清点的员工递账本、合同、名册和清单，不承担主要判断。

蒙太奇处理：

1. 谈判之间的蒙太奇应参考 v12 前的空间打开方式，利用不同光区、多媒体和群像动作形成影视快剪感。
2. 蒙太奇节点不局限于并购章节，可按时间检索并调用1915至1922年前后的南烟大事记。
3. 可用节点包括：1915年前后国货展获奖和“中国人请吸中国烟”；1915年上海商号来信求购但苦于无分销机构；1916年前后上海分公司开幕，南京、镇江、苏州等渠道打开；1917至1919年高阳大楼、上海制造厂和总管理部；1918年前后长城牌与国货广告；1919年身份危机、“敬告国人”、公开账目、脱离日籍、公开招股；1920年前后赈灾、义学、留学资助等社会责任信息。
4. 蒙太奇服务谈判主线，不另起资料展。每段蒙太奇最终都要把订单、账本、员工名册、广告、舆论和厂房压力递回谈判桌。

## 七、第三幕创作要求

第三幕采用三段式舞台结构：第一段由陈荣贵现实迁厂，承担清点人员、清点机器、催船和组织搬运的判断；第二段以前场简玉阶为主，后场迁厂动作持续，让观众看清他在主导权旁落、战火压境、外部拉拢和员工迁厂之间的精神困局；第三段回到重庆废墟复工，以机器重新响起完成合流。

第三幕不写简玉阶与陈荣贵现实会面。简氏家族的精神通过前场暗线、旧物、回声和第二幕承诺延续，不把简氏兄弟写成战时迁厂现场的现实指挥者。

人物动机应从战时现实出发：活下去、护住家人、让机器和物资不落入敌手、保存民族工业火种、在废墟中恢复生产。工人担忧“什么时候开工”可保留为复工阶段情绪，但需放在安全、撤离和家国处境之后。

## 八、人物与演员边界

1. 简照南、简玉阶固定为同一演员、同一角色贯穿。
2. 简铭石只作为第一幕历史人物出现；如同一演员第二幕继续上场，应换身份为董事/股东复合角色。
3. 陈荣贵为第三幕有史料依据的战时执行人物，承担汉口厂迁运、重庆制造厂建立和轰炸后复工的行动判断。
4. 员工角色按时代和场景拆开：初创员工、战时老员工、尾声青年员工等是不同角色身份，可以由同一演员分饰，但台本不能写成同一个员工跨越几十年。
5. 账房不作为贯穿配角强化。账本、清单、合同、档案等信息由南烟员工群像用动作承接。

## 九、后续审核清单

1. 每个关键情节需标注资料来源、时间位置和艺术加工程度。
2. 每个主要角色需区分明确原型、复合角色、虚构角色和画外音压力。
3. 每次谈判需明确对应1914年、1917年、1918年草案至1922年初继续磋商中的哪一轮史实。
4. 第二幕蒙太奇需核对1915至1922年前后各节点出处，不要把不同年份写成同一天发生。
5. 第三幕需重点核对陈荣贵、汉口厂迁运、重庆制造厂建立、轰炸后复工等战争时期社会动态。
6. 所有具体年份、地点、金额、机器数量、人员数量、品牌名称需在最终提交前回看原PDF页。
