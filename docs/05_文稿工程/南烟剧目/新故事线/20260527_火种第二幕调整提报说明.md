# 《火种》第二幕调整提报说明

整理时间：2026-05-27  
用途：供对上沟通、确认创作口径使用。  
相关会议记录：`导演会/20260527215320-转写_佳。的快速会议-逐字稿文本-1.docx`

## 一、本轮调整的总体思路

根据最新讨论，第二幕拟在现有历史事件版基础上继续调整。调整后，第二幕仍保留与外资并购谈判的主线，但压缩谈判篇幅，并把原来以多媒体蒙太奇呈现的红双喜发展、教育公益、早期工人运动等内容，转化为更具体的舞台事件。

这一版希望解决两个问题：

1. 保留舞台戏剧冲突。  
外资并购谈判天然形成对外压力，能够让“国人自强”“中国人做中国烟”“民族企业守住主体性”这些主题有明确对抗对象。

2. 回应客户对红双喜、红色基因和上实承接的关注。  
第二幕会强化红双喜、国货传播、上海厂、教育公益和早期社会思潮，使南烟的民族企业属性不只停留在谈判态度上，也能通过具体行动被观众看见。

## 二、为什么建议保留谈判线

1. 谈判是第二幕最清楚的对外冲突。  
如果完全取消谈判，第二幕容易变成品牌发展、公益行动和历史资料的串联，戏剧压力会变弱。保留谈判，可以让观众清楚感受到南烟面对的是外资强权、资本诱惑和市场控制。

2. 谈判能集中表达“守住主体性”。  
红双喜、上海厂、国货渠道和员工队伍都是南烟做大的结果。谈判桌上的问题，是这些成果最终归谁、听谁、为谁运转。它能把品牌、工厂、员工和公众信任统一到一个戏剧问题中。

3. 谈判便于舞台表达。  
同一张长桌可以反复出现，通过合同厚度、灯光、人物站位、简照南身体状态和画外音变化表现时间推进。外方不需要具象成角色，仍可用画外音、字幕、合同文本和打字机声完成压迫。

4. 本轮会缩小谈判比例。  
前两轮谈判只保留关键冲突和态度，不再过多铺陈条款。重点篇幅会交给红双喜、国货发展、教育公益、社会思潮和兄弟精神交接。

## 三、为什么“支持革命”部分建议先选取早期工人运动

目前资料中可关联“支持革命”“红色基因”的线索较多，包括孙中山革命事业、五四期间身份风波、早期工人运动、南烟职员同志会、五卅运动、后续党组织等。综合时间线、舞台效果和敏感度，建议本轮先以1921至1922年前后的早期工人运动和烟草行业风潮作为切入。

理由如下：

1. 时间更接近第二幕。  
第三轮并购谈判发生在1922年初。1921年英美烟草工人罢工、上海烟草工人会成立，以及1922年前后南烟职员同志会等线索，与第二幕时间距离较近，能与三轮谈判自然衔接。

2. 场景能落在上海厂和烟草行业。  
早期工人运动与烟草行业、上海工厂、职员同志会等空间关系明确，能与高阳大楼、上海制造厂、工人群像同场呈现，舞台上更容易形成社会空间。

3. 表达口径相对稳妥。  
可以写成南烟处在20世纪20年代初上海工人运动和国货自强的时代风潮中，表现南烟顺应社会进步、重视员工、支持国货和教育。这样的表达比直接写“南烟领导革命运动”更准确。

4. 能与兄弟精神交接形成呼应。  
当外资谈判进入尾声，舞台外部响起工人、学生、报童、国货消费者等时代声浪，简照南把守业责任交给简玉阶，能让交接从家族内部扩大为时代责任。

需要说明的是，1922年11月南烟职员同志会17条会谈具有戏剧性，但它也包含企业内部权益诉求和劳资关系，不宜未经确认就写成“支持革命”的主事件。当前可将其作为候选素材或背景风潮，等待领导判断是否进入正戏。

## 四、需要领导判断的敏感点

### 1. 五四期间“敬告国人”及日资日籍身份风波

资料中记载，五四期间竞争对手借简照南曾经加入日籍等问题攻击南烟为日资，南烟连续发表“敬告国人”，公开账目，邀请国人查阅，办妥脱离日籍手续，并公开向国人招股、改组公司。

该事件的戏剧价值很高：

1. 能表现南烟如何把国货身份从口号变成公开行动。
2. 能解释南烟为何从家族企业走向公众国货。
3. 能强化第三轮谈判时“这已是国人共同认购的南烟”的分量。

但该事件涉及日资、日籍、舆论攻击、执照危机等内容，建议请领导明确：

1. 是否可以在提交稿中直接写出“日籍”“日资”等词。
2. 是否可以写“脱离日籍手续”“公开账目”“公开招股”。
3. 若需弱化，是否统一改为“身份风波”“国货身份遭质疑”“公开自证”等表述。

### 2. “支持革命”具体写到什么程度

当前资料中可选方向包括：

1. 早年支持孙中山革命事业及相关捐助。
2. 五四期间国货运动和身份自证。
3. 1921至1922年前后的烟草行业工人运动。
4. 1922年南烟职员同志会和17条会谈。
5. 1925年五卅运动及南烟工人参与。
6. 后续党组织、妇女运动、劳工夜学等红色线索。

建议请领导判断哪些内容适合在本次15分钟情景剧中出现，哪些只适合作为资料储备或尾声多媒体一笔带过。

### 3. 五卅运动是否进入本节目

五卅运动时间在1925年，距离第三轮谈判已有数年，政治和情绪分量都较重。若进入正戏，会明显改变第二幕时间线和主题重心。当前建议暂不作为第二幕主事件，除非领导明确希望强化后续红色革命线。

## 五、当前建议的安全表达口径

1. 谈判线表达为“面对外资强权和资本控制，南烟守住民族企业主体性”。
2. 红双喜表达为“从烟标、柜台、工厂和公众信任中成长起来的国货品牌”。
3. 教育公益表达为“取之社会、用之社会的民族资本家责任”。
4. 支持革命表达为“身处时代风潮，回应国货自强、工人觉醒和社会进步”。
5. 五四身份风波在领导确认前，暂用“国货身份遭质疑、南烟公开自证”概括。
6. 不把南烟写成早期工人运动的直接领导者，不把内部权益会谈简单等同于革命行动。

## 六、拟向领导确认的问题

1. 第二幕保留并购谈判作为对外冲突骨架，是否认可？
2. 前两轮谈判压缩，第三轮谈判承担情感高潮和兄弟交接，是否认可？
3. 红双喜、教育公益、早期工人运动作为第二幕具体事件展开，是否认可？
4. 五四期间“敬告国人”及日资日籍身份风波是否可以写？可写到什么程度？
5. 支持革命部分是否优先采用1921至1922年前后的早期工人运动和烟草行业风潮？
6. 1922年南烟职员同志会17条会谈是否可以作为正戏事件，还是只保留为资料依据？
7. 五卅运动是否需要进入本节目，还是暂不写入？

## 七、提报用简短表述

本轮第二幕调整建议保留外资谈判，但将其压缩为串联性戏剧骨架。这样既保留对外冲突和舞台张力，也能避免第二幕变成单纯并购史。谈判之间及谈判之后，将红双喜发展、国货信任、教育公益和早期工人运动等内容转化为具体舞台事件，最终落到1922年前后符合时代背景的社会思潮和简氏兄弟的精神交接。

其中，“支持革命”部分建议先采用早期工人运动和烟草行业风潮作为切入，表述为南烟身处时代进步潮流、回应国货自强和工人觉醒。五四期间“敬告国人”涉及日资日籍身份风波，史实价值高但敏感度需要确认，建议请领导判断是否可写及具体表述尺度。
