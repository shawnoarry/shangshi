# 南洋烟草开场情景剧《火种》

## 剧情大纲

### 核心主题

本剧以南洋烟草早期发展中的三次关键危机为主线，讲述简氏兄弟和一代民族实业人在经营受挫、资本围猎、战火内迁中守住民族工业火种的故事。作品作为上实45周年特别活动中的语言类节目段落，以南洋烟草的祖业故事开启整场演出的历史情绪：一缕火从品牌初创处亮起，在商战重压中延续，在战火废墟中重新开机，最终汇入上海实业承接百年民族工业的时代脉络。

全剧围绕三个问题展开：败了，还办吗；做大了，还归谁；战火来了，什么必须留下。三次追问逐层加压，形成从东山再起、商战谈判到战时护火的戏剧推进。舞台上，烟标、账本、合同、机器和档案贯穿始终，让“国货”“实业”“祖业”落在人物选择和具体行动中。

### 剧情概述

1905年前后，年轻的简照南、简玉阶兄弟在香港创办南洋烟草，推出“白鹤”牌香烟。白鹤初入市场便得到街面关注，却因商标纠纷遭遇重创。巡理府前，价值2000多元的“白鹤”牌香烟被当众销毁。简照南站在烟箱前承下这份羞辱，也把工人、烟贩和买主的信任记进账本。

失败之后，兄弟陷入自责与挫败。舞台进入心理空间，外商指控、街面疑惧、工人追问和清算压力交叠而来。随后，清算仓库显形，机器、物料、债务和无人敢买的现实摆在眼前。简铭石以现实支持托住兄弟，街面民众以谨慎而真切的方式返场，给出对国货再起的期待。简氏兄弟由此选择东山再起。

随着南洋烟草在国货热潮、渠道扩张和上海设厂中逐渐壮大，外资巨头多次抛出收买、合并和控制权方案。第二幕以1914至1922年前后的多轮商战谈判为历史基础，压缩为三次谈判桌戏。每轮谈判之间穿插舞台蒙太奇和多媒体快剪，呈现订单增加、广告铺开、市场扩张、公开招股、工人队伍扩大。南烟越发展，合同越厚，诱惑越重，简氏兄弟的选择也越清晰：资金可再筹，工厂可再扩，企业的根交不得。

抗战时期，战火逼近，民族工业内迁成为保存实业火种的关键行动。第三幕由汉口拆机、江上转运、重庆重开工三段构成。前两段呈现空袭、撤离、家人安危、设备抢运、敌占风险和江路艰险，第三段集中落在重庆厂废墟中重新开机。人物在战火中完成精神统一：机器声微弱响起，工厂仍在，手艺仍在，家国的希望仍在。

尾声由两名南烟员工整理资料收束。搬迁清单、残留烟标和开工记录被放进同一个档案夹，青年员工问这些以后还有没有人看，老员工回答有人接着做，就有人看得懂。多媒体短促点到1980年南洋烟草产权关系归属上海市政府、1981年上海实业有限公司在香港成立。情景剧在“火还在”的人物落点中结束，后续歌舞接住这份情绪，进入百年祖业与上实45年承接的整体表达。

### 整体风格与结构

本剧为历史情景剧，整体气质厚重、紧凑、热烈。舞台表达采用现实场景、心理空间、舞台蒙太奇与多媒体信息层交替推进的方式。外部压力通过画外音、字幕、报纸标题、合同文本、警报声和冰屏影像呈现；舞台正面聚焦简氏兄弟、家族支撑、南烟员工群像、街面民众和企业内部选择。

全剧按12分钟左右体量组织，并与后半歌舞组成开场组合节目。结构为“引子 + 三幕 + 两段过场 + 尾声转歌舞”：

- 引子：白鹤初燃，巡理府前当众销毁，形成第一记冲击。
- 第一幕：从心理坠落到东山再起，回答“败了，还办吗”。
- 过场一：国货热潮、渠道扩张、上海升级，说明南烟为何进入更大的商战。
- 第二幕：三轮并购谈判与两段内嵌蒙太奇交替推进，回答“做大了，还归谁”。
- 过场二：多媒体延续南烟发展，报童轻动作带出战时新闻，商战压力转入战争压力。
- 第三幕：汉口拆机、江上转运、重庆重开工，回答“战火来了，什么必须留下”。
- 尾声：两名员工整理资料，完成火种落点，短桥段承接后半歌舞。

### 出场人物

1. 简照南：史料中有明确原型的主要人物。作为全剧主行动人物，他从初创时期的实业理想，走向失败后的自责，再到商战压力中的成熟选择。

2. 简玉阶：史料中有明确原型的主要人物。作为兄弟关系中的情绪力量，他更直接、更不服输，在第一幕和第二幕承担强烈的行动冲动与守业表达。

3. 简铭石：史料中有明确原型的主要人物。作为第一幕的重要支点，他以现实支持帮助兄弟走出清算困境，把失败后的情绪重新推回行动。

4. 南烟员工群像：有资料基础的复合角色，包括老员工、青年员工、管账员工、机修员工、搬运员工、重庆厂员工等。员工群像承担工厂现场、账目清点、设备搬迁、战时复工和尾声档案整理，不设置单一贯穿配角。演出时可由同一组演员按场景复用。

5. 董事/股东复合角色：第二幕内部冲突人物，分出主和、主战、务实三种声音，代表资金、扩厂、工资、债务和市场风险等现实考量。

6. 时代环境群像：报童、烟贩、商号伙计、商会代表、学生代表、广告宣传员、街市商贩、船工、搬运工等，承担时代背景、市场流动、国货气氛、战时转运和节目衔接。

舞台主要角色以简氏家族为核心。资料中另有明确姓名的历史人物，在后续台本中可弱化为合同署名、报纸字幕、画外音或背景信息。员工与环境群像以身份指代，演员可复用，舞台重心始终落在简氏家族和南洋烟草三次危机上。

### 场景设置

1. 香港码头/罗素街旧货仓外场：用于引子。市井声、搬货声、报童声和洋烟广告共同构成初创环境。

2. 巡理府前/白鹤销毁空间：用于引子的情绪转折。法槌声、外商画外音、字幕和火色灯光共同形成第一次重压。

3. 心理压力空间：用于第一幕前半段。冰屏、报纸标题、重叠人声和四面方位调度构成挫败感。

4. 清算仓库/员工账台空间：用于第一幕后半段。旧机器、账本、烟标、牌匾、物料箱、空烟盒和旧订单构成主要道具。

5. 国货发展蒙太奇空间：用于过场一和第二幕谈判间隔。演员调度、地图点亮、广告画翻转、烟箱传递、订单堆叠、工厂剪影共同呈现企业发展。

6. 谈判桌：用于第二幕。同一张桌子串联多轮并购谈判，通过合同厚度、灯光方向、人物站位和服装细节表现时间推进。

7. 战时工厂/码头/江面/重庆厂：用于第三幕。机器架、木箱、纤绳、船板、警报声和烟尘灯光完成多空间转换。

8. 重庆厂重开工/档案光：用于尾声。机器声完成情景剧精神落点，两名员工整理搬迁清单、烟标和开工记录，短桥段把情绪交给后半歌舞。

### 剧情大纲

#### 一、引子 ——【白鹤初燃】

舞台从香港码头或罗素街旧货仓外的热闹场景开始。汽笛、叫卖、搬货、报童声交织，年轻的简氏兄弟穿行其间。新做好的“白鹤”烟标被员工举起，烟贩说街面上有人开始问这个牌子。简玉阶兴奋地接过样烟，简照南望向远处洋烟广告，意识到真正的对手已经站在前方。

兄弟短对话点出初心：洋烟一船一船进来，中国人的钱一船一船出去；中国人自己造、自己卖、自己被国人信任，这件事有了超出生意的分量。舞台呈现初创的朝气与街面的信任。

氛围突然转冷。外商指控以画外音进入，冰屏出现法庭、商标、败诉、销毁等纪录片式字幕。巡理府前，员工本能护住烟箱，烟贩怕被牵连，简玉阶上前理论；简照南拦住他，把烟贩的欠账记进账本，又从员工手中接过一包“白鹤”，放进被勒令销毁的箱内。他站在烟箱前承下这一次羞辱。刚才那句“让国人信任”的理想，在这一刻变成他对员工、烟贩和街面买主的欠账与责任。

舞台以火色灯光写意映出白鹤标识的崩塌，字幕明确为“价值2000多元‘白鹤’牌香烟在港英当局巡理府前当众销毁”。引子结束时，简照南把残留的一张白鹤标识夹进账本，简玉阶看着空了的烟箱，微光接入第一幕。

#### 二、第一幕 ——【败后再起】

第一幕承接引子末尾的火色和账本。简照南将白鹤标识夹进账本后，舞台随即进入抽象心理空间。四面声音压上来，外商声音、报纸标题、烟贩低语、员工追问、街面议论彼此叠加。冰屏反复出现“败诉”“销毁”“清算”“谁还敢买”“谁还敢卖”等词。简玉阶几次冲向声音来源，又被一圈圈人影拦回；简照南抱账本站在中央，所有声音像一笔笔追来的债。

心理空间中，兄弟的挫败感逐渐具象化。简玉阶的不甘化作一声声“凭什么”；简照南的压力落到员工明天的饭碗、烟贩躲开柜面的眼神、街面买主被牵连后的沉默。前面“让国人信任”的理想倒转为最重的压力：信任一旦被打掉，受伤的是一个牌号，也是所有愿意试一试国货的人。

现实清算随后显形。管账员工报出机器、物料、债务和无人敢买的局面，员工守在旧机器旁，等一个明日去处。简铭石上场，把兄弟从心理空间拉回仓库现实。他先问机器在哪里、物料在哪里、债怎么还、人还剩几个，再把现实支持放上桌。败一次可痛，自己认输才真正断路。

街面民众在后半段返场。引子中的烟贩、商号伙计、报童和买主从心理空间的质疑声中走出，先是犹豫地把空烟盒、赊账条或旧订单放回桌上，随后有人低声说：“等你们再开张，我还摆出来。”也有人说：“这次没敢摆在柜面，下次你们把牌子做稳，我第一个问货。”这些声音带着市井里的谨慎、同情和不服气，与前半段的质疑形成对照。

这一幕由精神坠落走向重新开机。管账员工把账本推到简照南面前，简照南亲手划掉旧债、写下新章程；简玉阶掀开蒙住机器的布，员工围上来扶正机件；简铭石把现实支撑放上桌，烟贩和商号伙计把旧订单压在账本旁。钱、章程、印章、工具和订单落在同一张桌上。旧机器重新启动，机器声从断续到稳定，第一束暖光亮起。

#### 三、过场一 ——【火怎样烧起来】

机器声延续，转为国货热潮的节拍。报童举起国货新闻，商号递来订单，烟箱在演员手中传递，地图上点亮南洋、广州、上海、南京、镇江、苏州等节点。演员推转长桌，旧仓空间变成销售桌、广告桌和工厂桌，高阳大楼与上海制造厂剪影出现。

过场一完成从重新站起到进入更大商战的逻辑。南烟靠产品、渠道、广告和公众信任做出声势。越来越亮的地图和广告被报价数字覆盖。外商画外音从“指控”变为“合作”，长桌被推到舞台中央，进入第二幕第一轮谈判。

#### 四、第二幕 ——【和战之间】

同一张谈判桌串起三轮并购谈判，三轮之间留出两段发展蒙太奇。每一次谈判结束，舞台都短暂离开桌面，用舞台蒙太奇和多媒体共同呈现南烟在时间间隔中的发展：订单增加、渠道外扩、上海厂建设、广告传播、公开招股、员工队伍扩大。南烟越发展，外商的报价越高，内部的现实压力也越重。

第一轮，外商抛出收买条件，带着威胁也带着诱惑。南烟刚刚站稳，简照南以高价回应，拖住对方，也显示这门买卖没有廉价退路。董事担心资金，简玉阶压住怒气，管账员工递上现有资产和债务清单。简照南接过清单，放在合同旁。第一轮落在弱势中的机智。

第一段内嵌蒙太奇随即展开。谈判桌灯光压暗，报童从台口穿过，报出国货声势和各地订单。多媒体上，地图从香港、南洋继续点亮到广州、上海、南京、镇江、苏州；商号伙计把订单一张张传入工厂空间，员工推过烟箱，广告宣传员翻出新的烟画。舞台动作简洁，重点信息交给多媒体和节奏快剪。外商画外音在快剪末尾再次出现：销量、渠道、工厂、品牌都被他们看见了。

第二轮，外方提出更完整的合并条件。条件从单纯收买转向资产、品牌、经营权和市场控制。内部分歧被推到台前：联合带来资金和安全，守住主体性意味着继续面对巨头压力和未知风险。简照南面对扩厂、工资、债务和市场，简玉阶承担更直接的情绪表达。工厂空间在谈判桌边缘亮起，员工群像站在机器声里发问：“厂还归谁，明天听谁开机？”

第二段内嵌蒙太奇从“做大”转向“做大后的压力”。多媒体出现高阳大楼、上海制造厂、广告烟画、公开账目、招股信息和员工名册；报童声音从兴奋变得急促，商号订单与资金缺口交替出现。舞台上，员工数量增多，机器节拍更密，账页和订单在多人手中传递。简照南接过一页页账目，简玉阶看向越来越密的机器声。南烟的声势越盛，桌上的合同越厚。蒙太奇末尾，打字机声压过机器声，第三轮谈判的厚合同被放到桌中央。

第三轮，合同更厚，条件更诱人。外商声音提出资产合并、品牌保留、仍由简氏经营等看似周全的方案。董事/股东复合角色提出现实焦虑：厂要扩，薪要发，债要还，市场还在打。管账员工递上员工名册、机器清单和账本，简照南把它们压到合同旁；工厂空间里的员工群像停下动作，看向谈判桌。简照南合上合同，报价纸被账本压住，机器声重新盖过打字机声。

第二幕落在现实压力中的承担：南烟缺钱，根不能丢；国货活得艰难，主体性仍要守住。三轮谈判层层加码，把南烟越做越大之后面对的清醒选择推到台前：卖掉更容易，守住更艰难。

#### 五、过场二 ——【时代压下来】

第二幕合上的合同留在桌上，报童带着新报纸从侧台跑过。多媒体先延续南烟这一时期的发展：新厂、订单、广告、招股、员工队伍、国货声势继续滚动。舞台动作保持轻量，报童换报，员工翻页、推烟箱、递清单，人物正面戏暂时收住。

屏幕信息逐渐转冷。商业新闻被九一八、淞沪、卢沟桥、武汉告急、民族工业内迁等战时信息取代。报童第二次跑回舞台时，手中的报纸已是战报。灯光由暖转青灰，远处警报声进入。员工合上商业账本，打开设备清册；战时负责人走到桌前，只留一句：“账以后再算，先点人，再点机器。”第三幕从这句话进入。

#### 六、第三幕 ——【江上护火】

第三幕分为三段，前两段铺垫战争压力，第三段集中完成高潮。汉口拆机和江上转运摆出真实困境：空袭逼近、人员要撤、家眷要安顿、机器重、船少、时间紧，设备和物资还必须赶在敌占前离开。重庆厂战火中开工成为主戏，前两段积累的焦虑和疑问都在这一段得到回答。

第一段为汉口拆机。厂内一面维持必要生产，一面拆卸设备。警报声几次打断清点，有员工抱着家书问家里人怎么走，有人盯着庞大的机器问拖不拖得动，有人担心留下的物资落入敌手。战时负责人让众人先登记人员，再清点设备，把卷烟机、切丝机、蒸叶机、烟叶、卷烟纸、锡纸一项项报出。青年员工问：“人都不知道往哪儿躲，这些铁家伙还带得走吗？”老员工答：“人要走，机器也要走。人活下来，手艺才活；机器不落到敌人手里，厂才有明天。”

拆机由慌乱转为分工。员工把家眷名册、设备清册和装箱编号放在同一张桌上，妇女和年长员工整理随身包袱，青壮员工拆机、编号、装箱、找船、守夜班。舞台动作不追求激烈打斗，重点呈现战时秩序从混乱中建立。机器零件被一件件抬出，清单上的字被红笔划下，汉口厂的灯逐渐暗去。

第二段为江上转运。木船、纤绳、烟箱和机器零件构成码头与江面，船工说水道险，员工抱着清单核对箱号，机修员工护着零件，搬运员工说机器不能丢。多媒体提示长江上行、险滩、水道和迁运艰难。青年员工一度脱力，问“这么慢，真的赶得上吗”；老员工接过绳子：“慢也要走。机器走一寸，厂就活一寸。”拉纤号子与设备清单交替出现，行动紧张、台词克制，把压力一路推向重庆。

第三段为重庆厂战火中开工。场景转到重庆南岸弹子石，抢运来的设备刚刚拼起，灯刚亮，警报再响。轰炸后，厂房受损，机器残缺，员工从防空洞和废墟边陆续回到厂里。有人先数人，有人找伤员，有人翻清单，有人扶正残损机器。青年员工看着断裂的机件，说：“厂房这样了，还能开吗？”老员工翻开清单，报出还剩的设备和原料；战时负责人回答：“剩多少，就用多少。今天两箱也开，明天六箱也开。机器响一声，人心就稳一分。”

这一段把前两段的疑问统一起来。汉口拆机，是把生产能力从战火里抢出来；江上转运，是把这点能力送到还能站住脚的地方；重庆重开工，是把能力重新交还给战时中国。老员工接上一句：“厂房能塌，机器能坏，可中国人自己造的这口气，不能断。”众人重新接电、扶正机件、整理烟叶，机器声很弱，却在众人的注视中逐渐稳定。

第三幕的情绪落点是重庆厂重开机。战时负责人、老员工、青年员工和员工群像围着机器，胜利感来自“人还在、机器还在、厂还能响”这个共同看见的事实。南烟的火种在这一刻升为民族工业的火种：战火压下来，实业仍要转；城市被炸过，人心仍要稳；国货的名字仍要留在中国人自己手里。

#### 七、尾声 ——【火种交出去】

重庆厂的机器声逐渐稳定，情景剧在两名员工整理资料的对话中完成尾声。老员工把搬迁清单、残留烟标和开工记录放在同一个档案夹里。青年员工问：“这些以后还有人看吗？”老员工回答：“有人接着做，就有人看得懂。”冰屏出现白鹤烟标、设备清单、旧厂影像和后来的档案画面，形成一条从失败到护火的线。

多媒体短促点到：1980年南洋烟草产权关系正式归属上海市政府，1981年上海实业有限公司在香港注册成立，南洋烟草、永发印务、香港天厨等百年民族工业企业成为重要基础。信息点到即止，情景剧的结尾仍落在南烟员工身上：老员工把手放在重新响起的机器上，说：“火还在，家国的希望就还在。”

随后进入节目衔接。歌舞演员从机器声和灯光中进入，接过档案夹、火光或机器节拍。南烟人物退到侧后方，完成第一段语言节目；后半歌舞接续祖业群像与上实45年的整体表达。舞蹈承担承接与放大，情景剧的故事已经在“重庆重开工”与“火还在”中完成。

### 多媒体与舞蹈衔接

台词承担人物选择：兄弟的动摇与再起、简铭石的现实支持、谈判桌上的内部分歧、战时员工围绕人员撤离、设备搬迁和重庆重开工的判断、废墟复产后的家国精神落点。

多媒体承担时间、地点和复杂背景：白鹤纠纷、并购条款、股权比例、战争新闻、内迁路线、1980/1981上实承接信息。

舞台蒙太奇承担时间跨度和发展质感：国货热潮、商号来信、渠道铺开、广告烟画、街头推广、第二幕三次谈判之间的实力变化。

舞台动作承担可感的戏剧动作：巡理府前当众销毁、账本清算、机器重启、合同合上、谈判间蒙太奇、拆机装箱、逆水拉纤、轰炸后重开工、员工整理档案、节目衔接。

情景剧以情节和台词完成自身收束。后半歌舞承接情绪和视觉能量，作为第二个节目展开祖业群像。
