# 上药音乐剧新修节目说明

> 适用节目：上药音乐剧新修
> 更新日期：2026-05-28
> 当前原稿基准：`01_执行稿/00_原始稿件/【原版定稿】上药音乐剧剧本《展馆奇遇记》.docx`

## 一、目录定位

本目录用于承接上药音乐剧新修的完整工作链路。原稿、会议纪要、修订需求、结构大纲、分场台本、正式稿件、提交审核版和历史版本，均应在本目录内归档，不与其他节目混放。

## 二、当前目录结构

- `00_会议纪要/`：导演会、客户会、内部沟通纪要及反馈摘录。
- `01_执行稿/`：修订任务稿、执行口径、工作底稿。
- `01_执行稿/00_原始稿件/`：外部直接提供的原始剧本、旧定稿、扫描件。
- `02_资料索引/`：人物线、史料、节目背景、引用页码和核验说明。
- `03_方案储备/`：备选结构、改编方向、主题切口和题名储备。
- `04_新增资料/`：后续新增采访、网页、扫描资料和待整理文本。
- `05_灵感归档/`：阶段性废稿、片段灵感、弃用段落和降权方案。
- `06_剧本大纲/`：改编大纲、分幕结构、剧情梗概和展开版。
- `07_分场台本/`：分场正文、音乐段落、舞台提示和工作台本。
- `08_正式稿件/`：进入排练或内部确认阶段的正式文本。
- `09_提交审核版/`：当前对上流转、送审和提交版本。
- `10_版本归档/`：被替代版本和历史留档。

## 三、归位规则

1. 新收到的原始 Word、PDF、扫描件、旧定稿，先进入 `01_执行稿/00_原始稿件/`。
2. 围绕修订任务形成的内部口径、拆解说明和执行底稿，进入 `01_执行稿/`。
3. 尚处结构阶段的文档，进入 `06_剧本大纲/`；进入正文后转入 `07_分场台本/`。
4. 当前有效提交稿只保留在 `09_提交审核版/`；被替代版本移入 `10_版本归档/`。
5. 暂未采用但可能回用的思路，进入 `03_方案储备/` 或 `05_灵感归档/`。

## 四、命名建议

统一建议：

`YYYYMMDD_上药音乐剧新修_文件类型_主题或版本.ext`

示例：

- `20260528_上药音乐剧新修_修订大纲_v0.1.md`
- `20260528_上药音乐剧新修_分场台本_v1.docx`
- `20260528_上药音乐剧新修_提交审核版_v1.docx`
