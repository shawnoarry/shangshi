# 新增资料说明

> 用途：存放后续新增的原始资料、临时素材和已整理资料，避免散落在根目录或混入既有索引。

## 目录分工

- `01_原始资料/`：放新收到的 PDF、Word、图片、表格、音视频清单等原始文件。
- `02_待整理资料/`：放从原始资料中提取出来、尚未核验或尚未归入索引的文字、表格、摘录。
- `03_已整理入索引/`：放已经完成整理、可被写作调用的资料卡、页码表、图片清单、事实核验表。

## 中文文档要求

后续新建的资料说明、摘录、资料卡、索引表和核验表统一使用中文文件名、中文标题和中文正文。原始外部文件可保留原文件名；如需重命名，优先在前面追加日期和中文主题。

## 资料流转

1. 新资料先放入 `01_原始资料/`，保留原文件名或在前面加日期。
2. OCR、摘录、人工整理结果放入 `02_待整理资料/`。
3. 核验完成、可以支持写作后，整理成资料卡或索引表，放入 `03_已整理入索引/`。
4. 重要资料应同步更新 `docs/文档使用索引.md`；如果属于《上实百廿》或南洋烟草事实素材，也应补入 `docs/03_资料索引/` 相关文件。
