# 南洋烟草情景剧编剧指导

更新时间：2026-05-26  
适用范围：45周年大活动中“南洋烟草”语言类情景剧段落

## 一、项目定位

南洋烟草情景剧是45周年整体演出中的一个语言类节目段落，资料整理和大纲写作均服务于这一段落。它承担“以南洋烟草历史讲述上海实业精神源流”的功能，不能替代整台45周年活动的全部语言节目策划。

本节目时长控制在12分钟以内，核心表达集中在南洋烟草从民族实业困境中站起、在商战中守住自主、在战火中保存工业火种的精神线索。其他南洋烟草史料和相关设想仍保留在灵感素材库，供后续节目或版本替换使用。

## 二、版本与资料管理规则

1. 每次较大修改均新起版本，不直接覆盖既有文稿。
2. 原版md、历史版本、方案类文档保留，不删除、不合并，除非明确收到“合并”或“覆盖”的指令。
3. 提交版大纲、内部细化大纲、史实审核、编剧指导分开管理。
4. OCR文本可用于检索和初筛，正式引用、具体页码、关键史实需回看原PDF或可靠出处。
5. 当前资料库需保持可复用状态，后续如果替换情节，不应依赖单一大纲倒推资料。

## 三、文风规则

1. 提交版大纲使用事实陈述和剧情呈现语言，减少内部指导口吻。
2. 避免“不是……而是……”“不……而……”等对照式句型。直接说明人物状态、戏剧情境和情绪推进。
3. 避免将创作意图写成说明书式表达。舞台呈现、人物行动、台词冲突应承担解释功能。
4. 人物和场景描述保持简洁、准确、有画面感。提交稿应让阅读者看到剧情如何发生。
5. 历史人物台词不得写成未经出处支撑的“原话”。可写为戏剧化表达，并在审核文档中标注。

## 四、整体结构原则

当前方向以“三个关键历史节点+多媒体快剪”的结构推进：

1. 第一部分：东山再起  
   从白鹤牌受挫、清算压力、民众情绪承接到简氏兄弟重整国货信心。第一幕的戏剧重点是失败后的主体性和再起决心。

2. 第二部分：商战谈判  
   以外商并购谈判为核心戏剧场，呈现民族企业在资本、市场、身份舆论中的三次危机。三次谈判之间必须穿插舞台蒙太奇和多媒体，展示南烟发展，不能处理成连续三场会议。

3. 第三部分：战时内迁与开工  
   前两段呈现战争压力、迁厂困境和众人恐惧，第三段“战火中开工”作为精神统一和情感高潮。重点落在民族工业保存火种、守住家国希望。

4. 重要节点快剪  
   初创、国货运动、市场扩展、身份风波、战时转移、归入上海实业体系等节点可用多媒体呈现，也可进入人物对话。

## 五、人物与原型规则

1. 舞台主要角色以简氏家族为中心，简照南、简玉阶为核心人物。
2. 有明确史料支撑的人物需在审核文档中标注出处和时代位置。
3. 董事、账房、工头、工人、报童、商贩、学生代表、商会代表、船工、码头工等角色可作为复合角色使用。
4. 环境戏和舞台蒙太奇中的配角可使用身份指代，除非涉及资料中明确记载的历史人物。
5. 其他历史人物如外商谈判代表、商会人物、管理人员等，若不承担核心戏剧功能，可弱化姓名，减少史实负担。
6. 工人角色可适当增加，用于形成时代压力、集体行动和舞台气势。工人的台词需符合所处时代的生存处境。

## 六、史实与艺术加工边界

1. 白鹤牌受挫、商标诉讼、货物当众销毁、清理拍卖、叔父帮助、重新注册、外商并购谈判、身份风波、战时迁厂、重庆复工、归入上海实业体系等为可核验史实方向。
2. 火焰、心理空间、众人围声、报童串场、舞台蒙太奇、复合谈判桌、工人集体台词属于舞台化处理。
3. 所有艺术加工都需服务历史情绪和人物主体性。加工不能制造违背时代背景的人物动机。
4. 战争背景下，人物首先面对生死、家人、撤离、空袭、物资和敌占风险。单纯以“今晚开工”为矛盾起点，容易弱化战时真实压力。
5. 日本相关身份与舆论内容较敏感，可压缩为背景、多媒体信息或动作组，不作为第二幕主要台词冲突。

## 七、逐段创作要求

### 引子

引子承接白鹤牌被诉、货物被销毁的史实。重点表现简氏兄弟主动承受羞辱和责任，体现“中国人自己造、自己卖、自己被国人信任”获得了超出生意的分量。

“烧掉货物”若继续使用火焰意象，应明确作为舞台写意。史料支撑的是“当众销毁”，不是明确焚烧。

### 第一幕：东山再起

第一幕直接承接引子的情绪，先进入抽象心理空间，呈现两兄弟承受的质疑、挫败和压力。

清算、债务、机器物料、无人敢买等现实压力放在心理空间之后进入。后半段加入民众鼓励，商贩、学生、买主或报童可返场，与质疑声形成对照，推动简氏兄弟坚定发展国货、东山再起。

### 过场一

过场一以报童或轻动作串联多媒体，呈现南烟发展、市场拓展、国货声势。舞台动作轻，多媒体承担信息推进。

### 第二幕：三次谈判

第二幕的主戏是三次面对危机的谈判。谈判之间必须加入舞台蒙太奇和多媒体，展示南烟发展、扩张、招股、市场网络和国货舆论。

谈判桌上的角色可复合处理，但需避免让无史料依据的人物以“历史当事人”身份进入关键决策。工人代表若出现，更适合作为精神性、民意性或舞台化旁证。

日本身份风波和相关舆论可压缩处理，重点放在资本控制、自主经营、民族企业信用和国货信任。

### 过场二

过场二从南烟发展多媒体转入战时背景。可由报童短暂入场，完成从商战年代到抗战年代的时代换景。舞台动作保持轻量，避免变成独立动作戏。

### 第三幕：战时内迁与开工

第三幕三段演绎需有轻重。前两段引出战争压力、撤离困境、设备物资、家人安危和敌占风险。第三段“战火中开工”集中完成精神表达。

人物动机应从战时现实出发：活下去、护住家人、让机器和物资不落入敌手、保存民族工业火种、在废墟中恢复生产。工人担忧“什么时候开工”可保留为复工阶段的情绪，但需放在安全、撤离和家国处境之后。

第三幕的台词承担情感拔高。动作和舞蹈只能辅助氛围，不能替代情景剧的矛盾、选择和精神表达。

### 尾声与舞蹈衔接

尾声先完成情景剧自身的收束，再承接后续舞蹈节目。南洋烟草段落应有独立情感落点，舞蹈是下一节目衔接，不是情景剧的结尾兜底。

归入上海实业体系、成为改革开放窗口公司的信息可作为尾声字幕或简短旁白，完成历史纵深。

## 八、后续审核清单

1. 每个关键情节需标注资料来源、时间位置和艺术加工程度。
2. 每个主要角色需区分明确原型、复合角色、虚构角色。
3. 每次谈判需明确对应1914、1917、1918、1922中的哪一轮史实。
4. 第三幕需重点核对战争时期的社会动态，避免把和平时期工厂逻辑移入战时场景。
5. 所有具体年份、地点、金额、机器数量、人员数量、品牌名称需在最终提交前回看原PDF页。
