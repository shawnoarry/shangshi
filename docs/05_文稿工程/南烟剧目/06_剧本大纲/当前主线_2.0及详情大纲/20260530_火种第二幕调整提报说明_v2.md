# 《火种》第二幕调整提报说明 v2

整理时间：2026-05-30  
用途：根据最新反馈，更新第二幕对上沟通口径。  
适用范围：当前主线 2.0 之后的第二幕修订。

补充说明：后续如无新指令，当前主线相关提报口径以 `新提交/0530南洋烟草开场情景剧《火种》剧情大纲3.0.docx` 及 `新提交/0530第二轮修改简述.docx` 为最新标准；本文件用于同步其第二幕表述逻辑。

## 一、最新明确反馈

五四时期身份受质疑的部分完全隐去，不能作为剧情、台词和提报说明中的正面内容继续使用。

因此，以下表述全部停用：

- 身份风波  
- 国货身份遭质疑  
- 日资、日籍、脱离日籍  
- 敬告国人  
- 公开自证  
- 以公开账目、公开招股回应身份攻击

## 二、调整后的总体思路

第二幕仍保留三轮谈判作为对外冲突骨架，但第二轮谈判之后的中段，不再写成“身份危机—自证回应”的结构，而改写为：

1. 第一轮谈判之后、第二轮谈判之前，以一句“孙文盛赞南洋烟草，为国家挽回利权者也”点出市场与国货声势抬高；  
2. 1919至1920年前后国货风潮持续升高；  
3. 1920年北方旱灾、上海米荒把企业责任具体压上长桌；  
4. 教育义举、学生代表回响和工人觉醒声浪进入工厂空间；  
5. 第三轮谈判因此不只是一份合同之争，而成为“这门中国人办起来的实业还能不能继续由中国人自己作主”的立场之争。

## 三、为什么这样改仍然成立

1. 不削弱第二幕的情绪抬升。  
删掉身份风波之后，第二幕的中段仍可依靠国货风潮、1920年灾年中的社会责任和工人声浪完成情绪上扬，不会塌成资料平铺。

2. 反而更集中。  
原来这一段容易把观众带进复杂解释。改掉之后，第二幕的立意更集中在民族实业的成长、担当和主体性上。

3. 更利于舞台。  
“报童短促带出孙中山对南烟的评价、订单压上桌、1920年北方旱灾与上海米荒进入长桌、学生代表致谢、工人声音走进厂房”都比一场解释身份的戏更容易舞台化，也更便于和三轮谈判咬合。

## 四、提报时建议使用的口径

建议统一说明为：

“根据最新反馈，第二幕中涉及五四时期身份受质疑的部分已全部隐去。原先可能触及身份风波的段落，现改写为第一轮与第二轮谈判之间对国货声势的历史回声、1919至1920年前后国货风潮的延续、1920年北方旱灾与上海米荒所带来的社会责任，以及学生和工人声浪的进入。这样既避开敏感点，也保留了第二幕从商业发展走向时代担当的情绪抬升，并让第三轮谈判更有时代分量。”

## 五、对写作的直接要求

1. 不再出现身份辩白式场面。  
2. 不再把“公开自证”作为第二轮之后的主戏。  
3. 第三轮谈判前的桌面材料，改用订单、员工名册、机器清单、1920年赈济账本、平粜与米票线索、助学名单等内容。  
4. 第三轮谈判的高点，落在“企业主体性”和“民族实业责任”上，不落在身份解释上。  
5. 第二幕后半的社会责任段，宜落成可演场面，如清点赈济米粮、核对出货单、学生代表致谢、工人代表接续进入，不宜再写回纯资料陈列。  

## 六、建议一句话概括

第二幕中段由“身份风波”改为“时代风潮”，由“解释自己是谁”改为“承担自己该做什么”，由此把第三轮谈判抬升为一场关于民族实业根脉归属的选择。
