# 南洋烟草情景剧专项规划

> 口径说明：上实集团45周年是整体活动，本资料区主要服务其中一个“南洋烟草/百年祖业”主题语言类情景剧节目。当前阶段不以本资料反推整台活动流程，也不替代后续其他节目的策划。

## 一、节目定位

本节目建议定位为整体活动中的一个独立段落，时长控制在12分钟以内。它承担的功能是用南洋烟草这条百年祖业线，补足上实45周年叙事中的历史根脉、沪港连接和“实业报国”精神来源。

它不承担整台活动的总叙事，不负责覆盖上实所有产业板块，也不负责把后续节目全部串联完成。结尾只需自然接回“上实接过百年祖业、面向新时代焕新发展”，给整场活动留出继续展开的空间。

## 二、12分钟内的内容边界

建议只抓三到四个强情节点，避免把南烟120年讲成流水账：

- 1905年罗素街旧货仓：火柴、旧仓、机器、民族工业起点。
- 1906年至1909年白鹤失败与东山再起：第一次失败不是终点。
- 1919年身份危机与公开自证：账本、身份、公众信任，是语言类节目的高光。
- 1937年至1941年战时内迁：机器西迁、工人护火，适合形成情绪高潮。
- 1981年划归上实：作为尾声衔接，不宜展开成长篇上实发展史。

若时长紧张，优先保留“1905起火、1919自证、战时护火、1981承接”四个动作。其他内容如广告烟画、义学复旦、南洋酒店、国际化转型，可作为台词点到、影像闪回或灵感储备。

## 三、执行候选方案

优先候选：方案E“三次追问”。  
理由是冲突清楚、语言驱动力强、信息密度可控，适合12分钟内完成。三问分别是“能不能造”“你到底是谁”“往哪里去”，天然对应创业、身份危机、进入上实与祖业焕新。

备选候选：方案A“一封传真，三束火”。  
理由是情绪线最完整，能用1981年传真连接1905年火柴。使用时需压缩历史解释，把三束火处理成三个舞台动作，而不是三段讲解。

可局部吸收：方案H“火种西迁”。  
它适合做情绪峰值，但不宜单独承担完整12分钟结构。可以作为方案A或方案E中的第三段高潮。

可局部吸收：方案C“账本、机器、校舍”。  
它适合作为物件系统：账本承载信用，机器承载实业，校舍承载责任。若进入12分钟剧本，建议只选“账本+机器”，校舍作为柔性一笔带过。

## 四、灵感储备方案

以下方案保留在资料区，但暂不作为12分钟情景剧主结构：

- 方案B“一包烟的旅行”：物件线清楚，但容易拉长为串联型节目，也需要处理烟草消费属性。
- 方案D“双城回声”：概念贴合沪港关系，但12分钟内若城市拟人化过重，人物冲突会变弱。
- 方案F“高阳大楼的一夜”：空间质感好，适合做影像段落或单场戏，不适合作为当前主结构。
- 方案G“烟画翻开的一百二十年”：视觉潜力强，更适合歌舞/影像节目或视觉转场。
- 方案I“祖业焕新发布会”：适合现代尾声或后续策划，但作为情景剧主框架容易有汇报感。

## 五、建议的12分钟结构

1. 引子，约1分钟：1981年香港办公室传真声响起，文件中出现“南洋烟草”等祖业名称；声音转入1905年火柴声。
2. 第一场，约2分钟：罗素街旧货仓创业，简氏兄弟、工人、卷烟机建立“能不能造”的问题。
3. 第二场，约2分钟：白鹤失败后重整，灯灭又亮，说明南烟不是一路成功，而是在失败中再起。
4. 第三场，约3分钟：1919年身份危机，多声部质询“你到底是谁”，以公开账目、脱离日籍、国人招股回应。
5. 第四场，约2.5分钟：战时内迁，机器、工人、码头、警报形成群像，落点是“火种要留下”。
6. 尾声，约1.5分钟：回到1981年，上实接过百年祖业；一句话点到今天祖业焕新，不展开为整台活动总结。

## 六、后续写作原则

- 先写人物行动，再补历史信息。
- 每个历史点只保留一个舞台动作：点火、重启、摊开账本、搬走机器、接过传真。
- 避免把南洋烟草段落写成整台45周年活动总纲。
- 避免消费化呈现烟草产品，重点放在民族工业、企业信用、工人命运和祖业承接。
- 结尾要“接回上实”，但只接到本节目该承担的位置，不抢后续其他节目空间。
