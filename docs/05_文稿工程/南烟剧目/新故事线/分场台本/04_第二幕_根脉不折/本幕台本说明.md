# 第二幕 ——【根脉不折】

## 基准内容

以 0530 新提交版为准。第二幕从双喜被街头认出、各地订单涌来开始，保留1914年、1917年、1922年三轮谈判压力；第一轮与第二轮之间以“孙文盛赞南洋烟草，为国家挽回利权者也”作为短促背景音点出国货声势；第二轮之后不再展开五四时期身份受质疑相关内容，改写为1920年北方旱灾与上海米荒中的赈济义举、学生代表与教育责任、工人运动声浪和兄弟精神交接。
