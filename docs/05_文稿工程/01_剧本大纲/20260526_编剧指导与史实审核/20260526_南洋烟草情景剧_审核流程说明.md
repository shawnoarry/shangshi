4. 后续写正式台本时，再参考提交版成稿格式中的台词排版、舞台提示和多媒体提示。

## 六、审稿推荐流程

1. 打开v7 md或docx，先看整体剧情是否满足12分钟情景剧需求。
2. 对照编剧指导v2，检查项目定位、文风、结构、情景剧与舞蹈衔接边界。
3. 对照演员分配说明，检查角色是否过多、是否有配角抢主线、演员复用是否清楚。
4. 对照史实与情节审核，逐段标注A类史实、B类历史演绎、C类艺术加工、R类风险。
5. 回到资料索引和OCR页，核对关键年份、金额、人物、机构、机器数量、地点。
6. 正式提交前回看原PDF页，确认舞台字幕和旁白中的史实表述准确。

## 七、当前待注意点

1. v7是当前最新正文，后续修改应新起v8，不覆盖v7。
2. v5史实审核仍可作为基础，但v7进入台本阶段前应生成v7专项审核。
3. 账房已被降级为员工群像中的管账功能，后续不要重新强化为贯穿配角。
4. 尾声两个员工整理资料的对话已确定保留。
5. 舞蹈是后续节目承接，情景剧自身需要完成剧情和情感收束。
