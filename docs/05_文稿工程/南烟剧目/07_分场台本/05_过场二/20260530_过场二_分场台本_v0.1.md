# 南洋烟草开场情景剧《火种》

## 过场二 —— 分场台本 v0.1

【舞台呈现形式】

本段控制在约 45 秒。由第二幕商战长桌转入战争时间，信息主要交给报童、战报声、长桌翻面和冰屏标题。过场只做时代压力转换，不展开新剧情。

【角色需求】

报童、员工群像。

【多媒体及舞美方向】

长桌仍在中区。桌上的合同、订单、股东名册被迅速收拢，换成战报、诉状、迁厂函和设备清单。冰屏快闪五卅、省港大罢工、北伐、九一八、淞沪、武汉告急等时代标题，最终落定“南烟落入他人之手、上海厂毁于战火、汉口厂必须西迁”。

【剧本正文】

【第二幕末的机器声还在，打字机声逐渐变成急促报铃。报童从台口冲过，手里的商业报纸换成战报。】

报  童  新报！时局急变！战火逼近！

【冰屏快闪：五卅、省港大罢工、北伐、九一八、淞沪、武汉告急。员工群像撤下合同和订单，长桌翻面，战报与设备清单落桌。】

报  童  上海厂毁于战火，汉口厂奉令西迁！

【冰屏落定三行字：南烟落入他人之手；上海厂毁于战火；汉口厂必须西迁。】

【警报声压下。灯光转冷。中后区机器构件被推成待拆状态，木箱、纤绳、设备清册进入。】

员工群像  清点设备！找船！护住机器！
