# 百廿资料与新故事线需求对应说明

更新时间：2026-05-27  
工作位置：`docs/05_文稿工程/南烟剧目/新故事线/`  
对应问题：客户希望第二幕从“香港谈判、外商谈判”转向“红双喜这个牌子怎么诞生，后来怎样成为红色企业，怎样和上实红色基因连接”。

## 一、总体判断

《上实百廿》中“实业救国、捐赠公益、拓宽内陆市场、上海投厂”几条资料都能对应新需求。

其中最适合替换原第二幕的主线，是“红双喜从品牌诞生到上海与内陆市场站住”。这条线兼具品牌、国货、上海、工厂、群众信任和商业冲突，能回应客户对“红双喜”的关注，也能保留原来第二幕所需的戏剧张力。

“实业救国”适合作为全幕动机。“捐赠公益”适合作为价值补强和多媒体快剪。“上海投厂”和“拓宽内陆市场”适合承担第二幕主要戏剧行动。“红色企业”和“上实红色基因”适合通过后段多媒体、尾声档案交接或人物回望接上1950年代公私合营、1981年归入上实、1997年香港回归等节点。

## 三、对第二幕替换的判断

原第二幕的谈判线有史实基础，但客户反馈显示“外商并购谈判”距离他们关心的红双喜、红色企业、上实基因较远。

更适合的新第二幕，可定为：

**第二幕：红双喜怎样站上中国柜台**

这一幕的中心行动：

1. 上海来信：商号想卖南烟，却因上海没有分销机构和外资压力买不到货。
2. 柜台试卖：烟贩、商贩、报童、消费者形成市场现场，“双喜”从柜台下被递出来，到重新被摆上台面。
3. 调整产品：简氏兄弟和员工根据上海口味、包装习惯、价格、佣金重新安排“双喜”等品牌。
4. 抢滩上海：南烟在上海周边打开局面，苏州、南京、镇江等地连成网络。
5. 上海投厂：简照南拒绝让出东百老汇路厂房，高阳大楼建起，“双喜”“爱国”“飞艇”等品牌从上海制造厂源源不断出厂。
6. 价值升格：多媒体快剪赈灾、兴学、义学、劳工学校、早期党组织、公私合营、归入上实，让观众看到“红双喜”从国货品牌成长为红色祖业的一条长线。

## 四、可形成的三个故事方案

### 方案一：上海来信

核心事件：上海商号来信希望采购南烟产品，却指出上海没有分销机构。简氏兄弟决定把红双喜带进上海，建立渠道、调整包装、争取烟贩和消费者。

优势：
- 戏剧触发点明确。
- 能把红双喜、国货、上海、群众信任放在同一场戏里。
- 适合保留12分钟情景剧节奏。

适合结构：
- 开场：报童读信，烟贩怕外资压力。
- 中段：调查员回报170家小同行的压力，简氏兄弟决定降价让利、调整包装。
- 高潮：商号、烟贩、消费者一起让红双喜重新摆上柜台。
- 收束：多媒体显示上海分公司、高阳大楼、内陆网络。

### 方案二：高阳大楼

核心事件：南烟进入上海后必须建厂。对手试图高价阻止，简照南拒绝让出厂房。高阳大楼拔地而起，红双喜从牌子变成生产能力。

优势：
- “上海投厂”非常符合上实集团的关注点。
- 舞台视觉强，有旧货仓到大工厂的空间升级。
- 主角主体性清楚，简氏兄弟的选择有重量。

适合结构：
- 开场：上海柜台缺货，市场倒逼建厂。
- 中段：厂房争夺，对手以高价阻止。
- 高潮：简照南一句“厂房非别可比”，众人开工建厂。
- 收束：机器声、工人群像、红双喜烟箱出厂。

### 方案三：一箱烟，一份米，一盏灯

核心事件：红双喜和南烟的市场增长，转化为赈灾、平粜、资助留学、复旦简公堂、义学夜学等社会责任。品牌被群众信任，原因在于企业把所得回到社会。

优势：
- 情绪温度高。
- 能表现民族资本家的责任感。
- 适合连接“红色基因”的价值底色。

适合结构：
- 开场：销售增长，账房报喜。
- 中段：水灾、米荒、留学生、义学孩子交替出现。
- 高潮：一箱红双喜被交到柜台，一袋米被交到灾民手里，一盏夜学灯亮起。
- 收束：转入早期党组织、公私合营、上实承接的档案快剪。

风险：
- 主戏容易离开“红双喜诞生”本身。
- 更适合嵌入方案一或方案二，作为中后段价值升格。

## 五、推荐方向

建议采用“方案一 + 方案二”为第二幕主体，以“方案三”为价值升格。

推荐题眼：

**红双喜怎样从一枚烟标，走上上海柜台，走进中国人的信任。**

这一方向能同时完成四件事：

1. 讲清红双喜诞生与“双喜”品牌前史。
2. 把原第二幕的商战张力从谈判桌转移到市场、柜台和工厂。
3. 用上海投厂、高阳大楼、内陆销售网络接上上实观众更关心的上海线。
4. 通过赈灾、兴学、劳工教育、早期党组织、公私合营、归入上实的快剪，把“民族资本家”和“红色基因”连成一条可信的历史链。

## 六、需要注意的史实边界

1. 1906年是“双喜”创牌，1909年是“红双喜”诞生。两者可以写成连续演进，不能合并成同一年。
2. 早期红双喜的“红”，首先是品牌视觉和喜文化，不宜直接写成政治意义上的红色。
3. “红色企业”主要依靠后续史实支撑，包括早期党组织、工人运动、公私合营、归入上实、香港回归和在港中资企业责任。
4. 捐赠、兴学、义学、赈灾能支撑民族资本家的社会责任，适合为红色基因提供价值铺垫。
5. 上海投厂和高阳大楼是第二幕最强舞台事件。它兼顾品牌、市场、上海、工厂和群众，建议优先写入新大纲。
