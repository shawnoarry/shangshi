# 南洋烟草情景剧初步大纲

> 版本：v0.1  
> 定位：上实集团45周年整体活动中的一个“南洋烟草/百年祖业”语言类情景剧段落。  
> 时长说明：本稿为早期10至12分钟压缩口径下的初步大纲；当前情景剧已调整为15分钟左右，最新提交口径以v10和编剧指导v3为准。
> 核心思路：舞台重点演绎三个关键危机节点，其余历史节点通过多媒体快剪、旁白和人物对话补足。

## 一、结构判断

当前项目资料已经可以支撑一个初步大纲。时间轴、分段概述、母题创意和结构方案中，已经明确了南洋烟草从初创、复起、商战、并购压力、战时内迁到归入上实体系的主要脉络。

本版大纲不再把“初创、战时、内迁”作为三个并列节点，而是调整为三次更有戏剧冲突的危机：

- 东山再起：败了，还办不办？
- 外商并购：卖了，还是守住？
- 战时内迁：毁了，还能不能留下？

这三段分别对应经营失败危机、资本吞并危机和战争生存危机。它们之间有清晰递进：先是能否重燃，再是能否守住，最后是能否活下去。

## 二、主线表达

一句话主线：

一支火，三次差点熄灭：败局中重燃，诱惑前守住，战火里转移。

核心母题：

- 火种：从旧货仓火柴，到机器炉火，再到战时护火。
- 机器：从旧机器重启，到工厂生产能力，再到内迁中被保护的工业根基。
- 账本：从失败清算，到谈判桌上的价值判断，再到战时清点设备物资。
- 祖业：不是怀旧物件，而是后来进入上实体系的精神资产和组织能力。

## 三、舞台三段

### 序幕：火从哪里来

建议时长：约1分钟。

呈现方式：多媒体快剪加旁白。

内容：

- 少年简照南离家谋生。
- 航运受挫，转向卷烟制造。
- 1905年香港罗素街旧货仓点火建厂。
- “白鹤”品牌短暂打开销路后遭遇商标纠纷，被迫清理拍卖。

功能：

让观众快速知道南烟不是从胜利开始，而是从失败和被打压中进入正戏。

### 第一场：东山再起

建议时长：约2.5分钟。

时间依据：1906年失败至1909年复办。

戏剧问题：败了，还办不办？

舞台场景：

夜里，旧机器、账本、牌匾或残余烟标仍在。简氏兄弟面对失败后的清算：债务、机器、员工、名声、下一步去留。

人物动作：

- 清点账本。
- 看见被迫摘下的牌匾或烟标。
- 兄弟争论是回到旧生意，还是重新办厂。
- 最后旧机器重新启动，灯光由暗转亮。

表达重点：

东山再起不是热血宣言，而是在失败之后重新承担风险。这里要让“民族工业不能一蹶不复兴”的责任感变成具体行动。

可通过对话带入：

- 1905年罗素街旧货仓。
- “白鹤”折翼。
- 国货和民族工业的起点。

### 过场一：火怎样烧起来

建议时长：约1分钟。

呈现方式：多媒体快剪。

内容：

- 国货热潮。
- “双喜”“飞艇”等品牌。
- 进入上海前“有人愿买，却买不到”的渠道矛盾。
- 南京、镇江、苏州等周边渠道逐步点亮。
- 上海高阳大楼、制造厂、机器声。

功能：

把第一场的“重燃”推进到第二场的“被看见、被盯上、被收买”。观众需要知道：南烟已经不再是小厂，它有了市场、工厂和竞争价值。

### 第二场：外商并购谈判

建议时长：约3分钟。

时间依据：1920年至1922年前后外商推动合并或收买南烟。

戏剧问题：卖了，还是守住？

舞台场景：

一张谈判桌。桌上一边是资本条件、合并方案、市场压力；另一边是工厂机器声、员工身影、品牌和公众信任。

人物动作：

- 外商代表或资本中介提出条件。
- 南烟方面翻看账本、合同、工厂产量和员工名册。
- 谈判中穿插工厂机器声，提示这不是单纯买卖，而是一整套工业能力和人的命运。
- 最终不以高喊口号结束，而以“这不是一张卖价表”一类克制判断收束。

表达重点：

这一场的价值在于把“国货”从口号落到主体性。南烟面对的不是简单敌我，而是诱人的安全、资金和市场条件。戏剧冲突来自：安稳和坚持之间怎么选。

可通过对话带入：

- 与英美烟草竞争。
- 产品质量、分销网络、人才和烟叶收购等商战基本功。
- 1919年身份危机和公开账目可作为一句回望，不必展开成独立场。

### 过场二：时代压下来

建议时长：约1分钟。

呈现方式：多媒体快剪加音效。

内容：

- 1930年代经营低谷和股权变化。
- 战火逼近。
- 上海厂受冲击。
- 汉口厂进入抢救式搬迁状态。

功能：

把第二场的“守住主体性”推进到第三场的“守住生存可能”。这段要让观众感到：前面守住的东西，现在要被战争连根拔起。

### 第三场：战时内迁

建议时长：约3分钟。

时间依据：1937年至1941年战时内迁。

戏剧问题：毁了，还能不能留下？

舞台场景：

警报声中，厂房灯光摇晃。工人、账房、家属、管理者同时在场。有人拆机器，有人清点物资，有人催促船期，有人不舍离开。

人物动作：

- 拆卸机器。
- 清点设备、烟箱和原料。
- 码头装船。
- 江面行进。
- 重庆厂在轰炸声中重新开工。

表达重点：

内迁不是“搬家”，而是护住工业火种。机器不是冷物件，它背后是饭碗、品牌、生产能力和企业还能再开工的可能。

核心落点：

厂房可以毁，机器要走，火种要留下。

### 尾声：火交到哪里去

建议时长：约1分钟。

呈现方式：舞台回到1981年文件/传真/档案盒。

内容：

- 1980年南洋烟草产权关系正式归属上海市政府。
- 1981年划归上海实业。
- 文件中出现南洋烟草、永发印务、香港天厨等百年祖业名称。

表达重点：

不展开上实45年发展史，只点明：上实接过的不只是企业资产，而是一束穿过百年的火。为后续其他节目留下空间。

## 四、多媒体快剪承担的内容

适合放入视频快剪：

- 少年离家、航运受挫、1905初创。
- “白鹤”失败。
- 国货热潮、双喜、飞艇。
- 上海渠道战、高阳大楼。
- 广告烟画、报纸、街头宣传。
- 1919身份危机、公开账目。
- 义学、复旦、社会责任。
- 1981划归上实。
- 当代祖业焕新。

不建议全部做舞台正戏。即便当前调整为15分钟左右，信息仍需分配给多媒体快剪、群像动作和关键短句。

## 五、资料充分性判断

现有项目资料已经足够支撑：

- 初步大纲。
- 导演阐述。
- 人物关系设计。
- 早期压缩结构草案。
- 多媒体快剪内容表。

进入正式台本前，还需要补充或核验：

- 外商并购谈判的具体年份、参与方和关键表述。
- 战时内迁中汉口厂、重庆厂、设备运输等细节的原 PDF 页码。
- 可用于大屏的图片素材页码：旧货仓、机器、厂房、高阳大楼、烟标、内迁相关图片。
- 主要人物小传：简照南、简玉阶、简铭石，以及谈判/工厂/工人群像是否使用真实人物或复合人物。

因此，本阶段可以进入“大纲创作”；但正式剧本台词和导演案最好再做一轮原 PDF 核页。
