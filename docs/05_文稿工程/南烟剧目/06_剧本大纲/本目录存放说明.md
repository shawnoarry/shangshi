# 本目录存放说明

> 适用节目：南烟剧目
> 目录：`06_剧本大纲/`

## 存放内容

- `当前主线_2.0及详情大纲/`：存放当前已通过提报的 2.0 主基准大纲及其展开参考稿。
- 不完全属于某一单独版本分支的南烟统筹大纲。
- 跨 `历史事件版`、`新故事线`、`三幕谈判版` 的总梳理、对比稿、汇总版和综合结构稿。
- 后续如出现“南烟项目总大纲”或“多版本比对大纲”，优先放入本目录。

## 归位规则

1. 当前南烟剧目主基准以 `新故事线/提交审核版/0528南洋烟草开场情景剧《火种》剧情大纲2.0.docx` 为准，主基准复制件进入 `当前主线_2.0及详情大纲/`。
2. 与 2.0 配套的展开参考稿、第二幕梳理、提报说明和会议总结，可随主基准放在同一子目录内，便于后续台本调用。
3. 历史事件版、三幕谈判版等旧方向的大纲继续留在对应分支内部。
4. 进入正文写作后，再转入 `07_分场台本/`。
