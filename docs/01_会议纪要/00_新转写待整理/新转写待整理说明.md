# 新转写待整理

存放新会议、采访、沟通录音的自动转写原文或未整理记录。

## 放入规则

- 会议转写：放在这里，命名为 `YYYYMMDD_会议主题_原始转写.docx` 或 `YYYYMMDD_会议主题_原始转写.md`。
- 采访转写：放在这里，命名为 `YYYYMMDD_采访对象或主题_原始转写.docx` 或 `YYYYMMDD_采访对象或主题_原始转写.md`。
- 整理完成后，在 `docs/01_会议纪要/` 新建对应的整理稿，并在总索引中登记。

## 整理提醒

自动转写通常会有人名、机构名、年份和口语断句错误。进入正式稿件前，应先整理为工作纪要或采访纪要，再与原始音视频或相关人员复核。

后续整理出的纪要、采访稿和可进入文稿工程的内容，统一使用中文文件名、中文标题和中文正文。
