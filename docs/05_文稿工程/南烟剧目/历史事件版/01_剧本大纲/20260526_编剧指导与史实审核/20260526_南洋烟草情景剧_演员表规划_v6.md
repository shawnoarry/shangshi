# 南洋烟草情景剧演员表规划v6

更新时间：2026-05-26  
适用版本：提交版大纲v14及后续正式台本  
用途：按角色身份整理有台词角色、演员分饰关系和各幕人数峰值，控制演员调用人数在10人以内。

## 一、总体判断

演员表先按“角色身份”列清楚，再单独说明由哪名演员分饰。员工不写成“员工A类”“员工B类”等分类名，避免台本阶段把演员功能线误读成同一人物或同一类角色。

v14后，所有外籍强权不再作为可见角色登台。外商指控、并购压力和日寇拉拢统一由画外音、字幕、合同文本、打字机声、警报声和影像压力呈现，不占台上演员名额。演员调用人数的核心压力转向南烟内部、员工群像和第三幕战时行动。

当前大纲按角色身份拆开后，约有22个可识别台上角色身份，外部强权画外音另列为声音功能。真正需要重点开口的台上角色约16至18个，其余可由动作、字幕或无台词群像完成。结合导演会“演员可调用人数在10人内”的条件，建议采用9至10人版演员配置，并保留8人压缩方案。

简照南、简玉阶必须固定为同一演员、同一角色贯穿。简照南只进入引子、第一幕、第二幕，第三幕可用旧照、烟标、账本或声音回响形成精神在场。简玉阶在第二幕末完成对兄长的守业承诺，并在第三幕以年老形象进入精神暗线，面对外部拉拢画外音作克制回应；台本不安排简玉阶与陈荣贵现实会面。

员工角色按时代和场景拆开：初创员工、初创青年员工、第二幕工厂员工、战时老员工、战时青年员工、尾声老员工、尾声青年员工分别是不同角色身份。可以由同一演员分饰，但不能写成同一个员工跨越几十年。

## 四、8至9人压缩备选

如果最终只能调用8至9人，优先压缩环境角色，不压缩简氏兄弟、陈荣贵与尾声两名员工。

| 压缩动作 | 处理方式 |
| --- | --- |
| 外部强权 | 全部改为画外音、字幕、打字机声，不占台上演员。 |
| 董事/股东复合角色 | 由简铭石演员分饰，第二幕以服装和站位区分。 |
| 报童/送信人/负责清点的员工 | 可由同一演员分饰，过场用帽子、报纸、信封、挎包区分。 |
| 烟贩/商号伙计/买主/船工 | 合并给一名环境演员，按场景换小道具。 |
| 搬运员工/机修员工/重庆厂员工 | 由演员4、演员5和环境演员共同承担动作，不另给大段台词。 |

8人版最低配置为：简照南、简玉阶、简铭石/董事、初创员工/战时老员工/尾声老员工、初创青年员工/战时青年员工/尾声青年员工、清点员工/报童、烟贩/商号伙计/船工、陈荣贵。外部强权声音全部使用画外音。

| 第二幕 | 6至7人 | 简照南、简玉阶、董事/股东、负责清点的员工、第二幕工厂员工、报童/环境角色 | 多个董事、公众质询信息 | 外资压力用画外音；蒙太奇可打开舞台但开口人数要少。 |

| 第三幕 | 7人左右 | 简玉阶、陈荣贵、战时老员工、战时青年员工、负责清点的员工、送信人/船工、机修员工 | 搬运员工、重庆厂员工、家眷 | 正戏抓陈荣贵行动线，送信人可由报童或船工演员兼任，不增加演员人数。 |

## 六、大场面压缩建议

1. 引子码头和罗素街旧仓不增加独立台词角色。热闹感交给音效、报童短句、搬货动作和烟箱调度。
2. 第一幕街面民众返场只保留烟贩和一个买主或商号伙计发声，其余人用递空烟盒、放订单、看向机器等动作表达。
3. 第二幕内部意见不拆成三个独立董事。董事/股东复合角色说现实压力，简玉阶说守业情绪，简照南做判断。
4. 第二幕身份舆论、公开账目、招股信息优先交由报纸、字幕、报童短句和资料递送动作完成；商会代表、学生代表不再作为当前提交审核版的可见角色单列。
5. 第三幕不要让家眷、伤员、多个工人逐个发声。台词集中在简玉阶暗线、陈荣贵、战时老员工、战时青年员工、清点员工、船工或机修员工。
6. 江上转运的拉纤场面可以由歌舞演员或无台词群像补身体量，但有台词船工只保留一名。
7. 重庆重开工不新增医生、警卫等角色，陈荣贵和员工群像即可完成。

## 七、台本阶段建议

正式台本拆台词时，建议每句台词旁同时标注“角色身份”和“演员编号”，例如“初创员工/演员4”“战时老员工/演员4”“尾声青年员工/演员5”“报童/演员7”。这样可以明确角色与演员的区别，也便于导演按实际演员条件替换。

尾声两名员工可由演员4和演员5承担，但角色身份应进入新的档案整理时空。尾声青年员工是后辈提问者，不应写成第一幕初创青年员工延续至尾声；情感承接依靠演员气质、道具和台词呼应完成，不依靠人物年龄穿越。
