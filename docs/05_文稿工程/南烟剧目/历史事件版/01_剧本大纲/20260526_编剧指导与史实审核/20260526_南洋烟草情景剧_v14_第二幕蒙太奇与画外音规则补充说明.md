## 一、v14调整结论

1. 第二幕三次谈判仍固定为同一张谈判桌，谈判桌是冲突锚点。
2. 谈判之间的蒙太奇不围绕桌子展开，应利用台口、侧区、后区、高区光影、大屏和冰屏形成时间流逝与企业发展的快剪。
3. 蒙太奇资料不局限于并购章节，可按1915至1922年前后时间线调用国货展、上海渠道、上海设厂、广告传播、身份危机、公开账目、招股和社会责任节点。
4. 所有外籍强权不作为可见角色登台，统一用画外音、字幕、合同文本、打字机声、警报声和影像压力呈现。
5. 台上人物面对画外音时可面向观众回应，让观众以第一视角感受历史压力。

## 三、第二幕台本处理提醒

1. 三次谈判的时间推进由同一张桌的变化完成，不另设三个会议室。
2. 谈判间蒙太奇要打开舞台，形成影视快剪感，避免演员都围着桌子递文件。
3. 每段蒙太奇最终应把信息递回谈判桌：订单、账本、员工名册、厂房影像、广告舆论和招股信息都要成为下一轮谈判压力。
4. 简照南身体状态逐轮变化，第一轮站立周旋，第二轮坐下审阅，第三轮扶桌停顿；精神判断逐轮更坚定。
5. 简玉阶在第二幕末接过账本和旧烟标，完成对兄长“守住根脉”的承诺。

## 四、外部强权呈现规则

1. 不设置可见外商代表或日寇代表。
2. 外部强权声音可从观众席方向、上方、冰屏后方或环绕声进入。
3. 合同条款、报价数字、股权比例、威逼词语可由大屏或桌面投影显示。
4. 台上人物回应时面向观众，形成“观众正在承受这段历史压力”的第一视角。
5. 配音可分层：殖民地司法压力、外资资本压力、战时拉拢压力可用同一声线变体，也可用不同录音处理；均不占台上演员。

## 五、陈荣贵资料位置保留

第三幕陈荣贵资料仍主要见 `work/上实百廿.pages.jsonl` 第174至180页，对应《上实百廿》第九章“战时内迁”：

1. 第174至175页：“汉口厂抢救式搬迁”。
2. 第177至180页：“重庆厂在轰炸中开工”。

可用于支撑陈荣贵的资料点：

1. 1938年前后，汉口厂准备向宜昌、常德、长沙方向迁厂。
2. 厂长陈荣贵向市府呈函，请求派船运输卷烟机等设备。
3. 南烟汉口厂在1938年9月30日全面停工，将机器和原材物料向重庆装运。
4. 重庆制造厂建立后，厂长由汉口厂厂长陈荣贵担任。
5. 重庆厂遭轰炸后，陈荣贵报告过警报停工、过江办事困难等现实情况。
6. 1939年9月28日，陈荣贵给宋子文的信中提到遭空袭后生产从每天2箱增加到6箱。
