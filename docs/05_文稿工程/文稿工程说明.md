# 文稿工程说明

> 用途：存放后续正式撰写、迭代和归档的剧本稿件。  
> 建议命名：`YYYYMMDD_节目名_稿件类型_第几版.md`，例如 `20260524_南洋烟草开场情景剧_分场台本_第01版.md`。

## 中文文档要求

当前文稿工作统一使用中文文档：文件名、标题、正文、版本说明和修改记录均使用中文。英文专名、原文引用、软件路径、文件扩展名和必须保留的外部资料原名除外。

## 目录分工

- `01_剧本大纲/`：放正式大纲、剧情梗概、幕场结构、人物关系表。
- `02_分场台本/`：放按场次拆分的台本、台词稿、舞台动作稿。
- `03_正式稿件/`：放准备对外流转、提交导演组或进入排练的正式版本。
- `04_版本归档/`：放已被新版本替代但仍需保留追溯的旧稿。

## 使用规则

1. 新稿先放在对应工作目录，不直接覆盖旧稿。
2. 文件名保留日期和中文版本号，便于追溯。
3. 形成正式流转版本后，复制或整理到 `03_正式稿件/`。
4. 被替代的旧版本移入 `04_版本归档/`，文件名不要改到失去版本信息。
5. 涉及年份、人名、机构名、金额等事实，提交正式稿前回到 `03_资料索引/` 和原 PDF 核验。
