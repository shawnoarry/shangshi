# 上实小镇节目说明

> 适用节目：上实小镇相关节目
> 更新日期：2026-05-28
> 当前状态：模板库已建立，待后续文件归位。

## 一、目录定位

本目录用于承接上实小镇方向的完整文稿链路。后续会议纪要、节目需求、资料索引、结构方案、剧本大纲、台本、正式稿件和提交审核版，均在本目录内独立归档。

## 二、当前目录结构

- `00_会议纪要/`：导演会、客户会、内部沟通纪要及反馈摘录。
- `01_执行稿/`：执行口径、任务拆解、工作底稿。
- `01_执行稿/00_原始稿件/`：外部直接提供的原始文案、旧稿、扫描件。
- `02_资料索引/`：人物、地点、时间线、史料或案例线索及核验说明。
- `03_方案储备/`：备选结构、题目、叙事切口和替代方案。
- `04_新增资料/`：后续新增采访、网页、照片说明、待整理资料。
- `05_灵感归档/`：片段灵感、弃用方案、过程废稿。
- `06_剧本大纲/`：剧情梗概、结构版、分幕版和展开版。
- `07_分场台本/`：分场正文、舞台提示、工作台本。
- `08_正式稿件/`：进入排练或内部确认阶段的正式文本。
- `09_提交审核版/`：当前对上审核、汇报和送审文件。
- `10_版本归档/`：被替代版本和历史留档。

## 三、归位规则

1. 新收到的导演会文件、聊天整理和反馈汇总，进入 `00_会议纪要/`。
2. 外部原始来稿或未改动底稿，进入 `01_执行稿/00_原始稿件/`。
3. 节目需求拆解、执行说明和内部工作稿，进入 `01_执行稿/`。
4. 结构阶段材料放入 `06_剧本大纲/`，正文阶段材料转入 `07_分场台本/`。
5. 当前有效提交稿只保留在 `09_提交审核版/`；被替代版本移入 `10_版本归档/`。

## 四、命名建议

统一建议：

`YYYYMMDD_上实小镇_文件类型_主题或版本.ext`

示例：

- `20260528_上实小镇_剧情大纲_v0.1.md`
- `20260528_上实小镇_导演会会议纪要.md`
- `20260528_上实小镇_提交审核版_v1.docx`
