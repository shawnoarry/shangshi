# 过往项目格式案例

> 用途：保存可参考的提交版大纲和成稿格式。这里只参考栏目、篇幅、排版和对上提交语气，不参考项目内容。

## 目录

- `提交版大纲格式/`
  参考剧情大纲、讲述大纲、音乐剧大纲的栏目组织。常见结构包括：核心主题或核心主旨、内容梗概或剧情概述、整体风格与结构、角色、剧情大纲。

- `提交版成稿格式/`
  参考正式成稿中的角色行、台词排版、舞台提示、多媒体提示和版本呈现方式。

## 使用边界

1. 本目录不作为南洋烟草史实依据。
2. 不从案例中借用人物、情节、项目表达或专有内容。
3. 新建提交版大纲时，可以参考其栏目顺序和段落颗粒度。
4. 新建正式台本时，可以参考其台词、舞台动作和多媒体提示的排版方式。
