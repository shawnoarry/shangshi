**南洋烟草三幕谈判版情景剧《和战之间》**

**剧情大纲**

**核心主题：**

本剧以南洋烟草1914年至1922年前后的三轮并购谈判为主体，讲述简照南、简玉阶兄弟在民族企业迅速壮大后，面对外资收买、合并方案、公众质疑和股东分歧时，如何守住企业主体、国货信用和民族工业根脉。

本剧从《火种》v21第二幕独立发展而来，剧情核心保持“做大了，还归谁”的追问。舞台以同一张谈判桌串联三轮谈判，谈判之外用国货热潮、商号来信、渠道扩张、身份自证和公开招股构成时间流动，使观众看见南烟拒绝合并的底气，来自市场、公众、员工和民族信用之间的艰难选择。

**剧情概述：**

南烟在“白鹤”失败后重新站起，凭借“双喜”“飞艇”等品牌和国货热潮打开市场。国货展获奖、各地商号来信、上海渠道需求和新厂扩张，让南烟从香港走向更大的中国市场。就在南烟逐渐成为外资巨头必须正视的对手时，第一轮收买试探到来。1914年，外部画外音以100万港元提出收买，并以“另有对付手段”施压。简照南没有正面冲撞，以资产与商誉共值300万港元周旋，暂时保住南烟的未来。

南烟继续扩张，上海、山东、浙江台州等地的订单和来信不断增加。1917年，第二轮谈判带着更完整的合并方案压到桌上：1000万港元作价、对方认购六成、仍由简氏兄弟经营、二十五年竞业限制、秘密合并等条款层层摊开。董事和股东有人看见资金与扩厂机会，有人担心品牌、经营权和国货名义被一步步收走。简玉阶怒意爆发，指出这类合并无异屈降；简照南把账本压在合同旁，让在场众人面对同一个问题：若国货只剩一个壳，南烟还算谁的南烟。

第二轮僵持后，南烟又遭遇1919年前后的身份危机。报纸质询、吊销执照、日资攻击接连逼到台前。简氏兄弟公开账目、办妥脱离日籍手续、连续“敬告国人”、公开向国人招股，并改组为南洋兄弟烟草股份有限公司。南烟在这一段中从简氏家族企业走向公众国货，股东名册、员工名册和账本一起落到谈判桌上。

1922年，第三轮谈判在更厚的合同前展开。遮佛士带来新公司方案：双方产品统一经营，制造厂按成本加一成佣金代制，新品牌共同所有，烟叶采购等环节避免竞争，资产比例看似优厚，南烟仍可由简氏兄弟管理。桌边分歧真正加剧：有人担心拒绝后失去资金和市场，有人以销售与盈利成绩证明南烟仍能继续打下去，也有人指向公开招股后的股东名册，提醒这已是国人共同认购的企业。

简照南此时病容已显，仍扶桌站起。他不再谈价码，只问牌子归谁、机器听谁、国人认购的到底是谁的南烟。简玉阶扶住哥哥，接过白鹤烟标和旧账本。第三轮最终落在合同合上，报价纸被账本压住。简玉阶向哥哥承诺：“哥，厂可以再难，根不能交。我守着。”全剧收在“做大了，还归谁”的回答上：南烟可以谈生意，可以算账，可以借众人之力扩张，但根脉不能交出去。

**整体风格与结构：**

本剧为一部商战历史情景剧，整体基调紧凑、克制、压迫感强。舞台空间以一张谈判桌为核心，三轮谈判固定在同一张桌上进行；每轮通过合同厚度、桌面文件、人物站位、服饰细节和简照南身体状态表现时间推进。

谈判之间的过场不围绕桌边完成，改用台口、侧区、后区、高区光影、大屏和冰屏展开舞台蒙太奇，呈现南烟发展速度、市场需求、国货声势和公众审视。外部资本压力不设置可见外籍角色，统一以画外音、合同文本、字幕、报价数字和打字机声呈现。

结构为“引子 + 三幕谈判 + 两段蒙太奇 + 尾声”：

1. 引子：南烟已重新站起，国货热潮和市场订单把它推向更大商战。
2. 第一幕：1914年收买试探，南烟以300万港元估值周旋。
3. 蒙太奇一：商号来信、渠道扩张、上海设厂，南烟被国人需要。
4. 第二幕：1917年完整合并方案，合并条款从价格转向主体权。
5. 蒙太奇二：1919年身份危机与公众自证，南烟成为公众国货。
6. 第三幕：1922年新公司方案，股东分歧和兄弟托付达到高潮。
7. 尾声：合同合上，账本留在桌上，完成“守住根脉”的回答。

**出场人物：**

1. 简照南：南烟创办者之一。第一轮稳住局面，第二轮追问企业主体，第三轮以病体完成最后判断。他的身体逐轮变弱，精神逐轮坚硬。

2. 简玉阶：南烟创办者之一。第一轮几次想冲出质问，第二轮正面爆发，第三轮压住怒气，扶住哥哥并承接守业托付。

3. 董事/股东复合角色：承担内部现实压力。可由一名演员或两名演员分饰不同立场，分别代表资金焦虑、发展信心和公众责任。

4. 负责清点的南烟员工：递送账本、合同、员工名册、机器清单、股东名册和商号来信，是信息进入谈判桌的主要行动者。

5. 报童/商号伙计/公众代表：承担国货热潮、商号来信、报纸质询、公开招股等蒙太奇信息，可由同一演员换小道具分饰。

6. 外部资本画外音：呈现三轮收买和合并压力，不作为可见角色登台。

**场景设置：**

1. 谈判桌：全剧核心空间。第一轮桌面简洁，第二轮合同变厚，第三轮文件最满。

2. 国货蒙太奇光区：用于呈现国货展、商号来信、订单传递、地图点亮和上海制造厂剪影。

3. 公众自证光区：用于呈现报纸质询、“敬告国人”、公开账目、脱离日籍、公开招股和改组公司。

4. 大屏/冰屏：用于合同条款、报价数字、报纸标题、地图节点、股东名册和企业发展影像。

5. 档案尾声空间：长桌回到安静状态，合同、账本、白鹤烟标、股东名册和员工名册留在台面，完成精神收束。

**剧情大纲：**

**一、引子 ——【做大了】**

舞台在机器声和国货广告声中开场。报童喊出国货展消息，商号伙计递来订单，南烟员工推着烟箱从侧区穿过。大屏上地图点亮香港、广州、上海、山东、浙江台州等节点，烟箱上依次出现“双喜”“飞艇”等烟标。

简照南和简玉阶站在长桌旁，看着订单和账本逐渐堆高。简玉阶带着兴奋说，街面终于有人主动问南烟的货。简照南看着越来越亮的地图，提醒弟弟：货卖得越远，桌上的账越重，盯着这张桌子的人也会越来越多。

机器声渐渐被打字机声覆盖。大屏上的地图被报价数字压住，长桌转为谈判桌。外部资本画外音第一次出现：“合作”“收买”“一百万港元”。第一轮谈判开始。

**二、第一幕 ——【一百万和三百万】**

1914年，桌上只有一份薄薄报价纸和一份资产清单。外部画外音给出100万港元收买南烟，又补上一句：若收买不成，另有对付手段。董事/股东复合角色看向简照南，担心外资接下来会用价格战、渠道战继续压迫南烟。

简玉阶听见威胁，半步起身，想正面质问这究竟是合作还是夺牌。简照南按住他的手，没有让情绪抢先。他让员工把机器、库存、商标、商誉和订单清单递上来，逐项摊开，随后提出南烟资产与商誉共值300万港元。

外部画外音停顿，报价纸被账本压住。简照南没有赢得轻松胜利，只是用一个更高估值拖住了对方，也给南烟争回继续发展的时间。第一幕结束时，简玉阶仍不服气，简照南告诉他：这一桌上，能守住一天，就多一天机会。

**三、蒙太奇一 ——【国货有人等】**

谈判桌灯光收窄，侧区和后区依次亮起。报童举起国货展消息，商号伙计把山东、上海、浙江台州等地来信递给南烟员工。大屏出现“上海无经售机关”“愿代推销”“中国人请吸中国烟”等关键词。

演员用快剪式调度完成订单传递：烟箱从台口到侧区，再到后区；地图节点从山东亮到上海，再亮到浙江台州；广告纸和烟画在空中翻转，高阳大楼和上海制造厂剪影渐渐出现。

这段蒙太奇让观众看见，南烟被资本盯上，是因为它已经被国人需要。蒙太奇末尾，员工把一摞新订单放回谈判桌，第二轮合同随即压上来。

**四、第二幕 ——【合并的壳】**

1917年，第二轮谈判开始。合同变厚，画外音从收买转向完整合并方案：南烟作价1000万港元，股份分为一万股，对方认购六成；合并后仍由简氏兄弟经营，总经理和协理人选照旧；南烟须在二十五年内不得另营竞争业务；合并可以秘密进行，对外广告竞争仍可维持表象。

董事/股东复合角色出现分歧。现实派认为条件优厚，可以换来资金和扩厂机会；另一种声音担心对方拿走六成股权后，牌号、市场和经营权会逐步被收走。负责清点的员工把员工名册和机器清单放到合同旁，让桌上的争论从数字落到人和机器。

简玉阶终于压不住怒气，正面回应：“名为合并，实则无异战败国之屈降。”这一句之后，桌边短暂沉默。简照南没有立刻拍桌，他把账本压在合同旁，看向董事和股东，问：若国货只剩一个壳，南烟还算谁的南烟。

第二幕的高潮由桌面文件的次序完成：合同在上，账本压住合同，员工名册和机器清单压在旁边。南烟的主体既包括牌号，也包括机器、员工、渠道和国人对国货的信任。

**五、蒙太奇二 ——【给国人看】**

第二轮僵持后，舞台进入公众自证。报童带着报纸冲入，报纸标题从“质询”到“日资”再到“吊销执照”。公众代表在侧区发问：南烟到底是谁的企业，国人凭什么再信它。

简氏兄弟没有退到幕后。员工群像把账册一页页举起，大屏出现“敬告国人”“脱离日籍”“公开账目”“公开招股”“改组公司”。负责清点的员工把公开账目放到桌上，商号伙计翻看，公众代表退后一步，股东名册随后落到谈判桌正中。

简照南可在这一段末尾说：“既已公之于国人，就不能负国人。”这句话把南烟从家族企业推向公众国货，也把第三轮谈判的责任压到更重。

**六、第三幕 ——【到底是谁的南烟】**

1922年，第三轮谈判开始。桌面最满：1918草案、1922新方案、股东名册、员工名册、机器清单、旧账本和白鹤烟标都在桌上。简照南扶桌起身，身体明显停顿。简玉阶先扶住哥哥，想替他说话，简照南轻轻按下弟弟的手。

外部画外音逐条压下：双方产品统一由新公司经营，双方制造厂与新公司为代制造关系，按成本另加一成作为佣金，新品牌为新公司共同所有，烟叶采购等方面避免竞争，南烟仍由简氏兄弟管理。大屏上报价数字不断闪现，资产比例看似越来越优厚。

桌边分歧真正加剧。现实派担心拒绝后失去资金和市场；信心派拿出销售与盈利成绩，认为南烟仍能继续打下去；公众责任派指向股东名册，提醒这已是国人共同认购的企业。简照南听完所有声音，没有再谈价码，只问：牌子归谁，机器听谁，国人认购的到底是谁的南烟。

这一问把第三轮推到高潮。简照南看向简玉阶，也看向整张桌子。简玉阶接过哥哥递来的白鹤烟标，压在合同边上。白鹤失败、东山再起、合并压力、公众自证和未竟之事，在这个动作里一起交到弟弟手上。

外部画外音仍在抬高报价，打字机声不断逼近。简照南把旧账本推到简玉阶面前，又把合上的合同推回桌心。简玉阶望向哥哥，低声回答：“哥，厂可以再难，根不能交。我守着。”随后他抬眼面向桌边众人，把账本压住报价纸。

机器声重新盖过打字机声。第三幕收在合同合上、账本压住报价纸、股东名册仍留在桌上的画面。南烟没有离开商战，也不会从此没有艰难，但它在这一刻回答了“做大了，还归谁”。

**七、尾声 ——【根不能交】**

桌面安静下来。外部画外音退去，只剩机器声和翻账本声。负责清点的员工把合同、账本、白鹤烟标、股东名册和员工名册整理在一起。报童从侧区经过，手中的报纸从质询南烟转为国货广告和新的市场消息。

简照南坐下，简玉阶站在他身旁。董事/股东复合角色没有再争论，众人看向桌上被账本压住的报价纸。尾声不做胜利欢呼，只保留一种沉稳的回答：南烟可以谈生意，可以算账，可以借众人之力扩张，但牌子、机器、员工和国人认购的信任，不能交出去。

灯光最后落在白鹤烟标、账本和股东名册上。机器声稳定，全剧结束。

**后续展开提示：**

1. 本版本只展开《火种》v21第二幕，不进入白鹤失败全过程和战时迁厂全过程。
2. 如需独立成15分钟左右节目，可适当加厚蒙太奇一和蒙太奇二，让国货市场和公众自证形成更清晰的情感铺垫。
3. 三轮谈判的外部压力均由画外音和合同文本完成，不设置可见外籍谈判角色。
4. 角色人数可控制在6至8名演员以内：简照南、简玉阶、董事/股东、负责清点的员工、报童/公众代表、商号伙计/环境角色，外部画外音另计。
