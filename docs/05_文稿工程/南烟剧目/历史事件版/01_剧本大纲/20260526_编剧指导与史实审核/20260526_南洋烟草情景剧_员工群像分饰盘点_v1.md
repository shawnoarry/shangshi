# 南洋烟草情景剧员工群像分饰盘点v1

更新时间：2026-05-26  
适用版本：提交审核版及后续台本展开  
用途：单独梳理“南烟员工群像”中各个角色身份与演员复用关系，避免把员工写成一个笼统类别，也避免把同一员工误写成跨越数十年的固定人物。

## 一、总体判断

员工群像应按“角色身份”拆开，演员复用另行标注。剧中真正需要固定为同一人物、同一演员贯穿的，是简照南、简玉阶等核心历史人物；员工线更适合按时代和功能切换身份。

当前版本中，员工角色大致分为四类：

1. 工厂现场角色：初创员工、初创青年员工、第二幕工厂员工。
2. 信息递送角色：负责清点的员工、设备清点员工、账本和名册递送者。
3. 战时行动角色：战时老员工、战时青年员工、机修员工、搬运员工、重庆厂员工。
4. 尾声传承角色：尾声老员工、尾声青年员工。

其中，“负责清点的员工”建议固定一名演员承担信息功能，但角色身份可随时代调整为清点员工、文书员工、设备清点员工等，不必写成同一个人从初创一直在场。战时老员工、战时青年员工需要在第三幕内相对固定，因为他们承担观众理解迁厂和复工情绪的工人视角。尾声两名员工保留为独立的档案整理角色，可由战时老员工和战时青年员工的演员分饰。

## 三、必须固定与可以分饰

必须固定的员工类功能：

1. 清点/文书功能演员：建议固定一名演员承担账本、合同、名册、设备清册等信息递送，便于观众识别“资料和现场事实”的来源。
2. 战时老员工：第三幕内固定，承担老工人经验和迁厂情绪。
3. 战时青年员工：第三幕内固定，承担青年视角和行动转变。
4. 尾声老员工、尾声青年员工：尾声内固定，保证收束对话清楚。

可以分饰的员工类角色：

1. 初创员工、第二幕工厂员工、战时老员工、尾声老员工可由同一演员换装分饰。
2. 初创青年员工、战时青年员工、尾声青年员工可由同一演员换装分饰。
3. 搬运员工、机修员工、重庆厂员工可由同一名动作型演员或群像演员分饰。
4. 报童、送信人、船工属于环境/转场角色，可与员工群像演员灵活复用，但台本中应另列角色名。

## 四、10人以内演员配置建议

| 演员编号 | 员工相关分饰 | 使用重点 | 注意事项 |
| --- | --- | --- | --- |
| 演员4 | 初创员工 / 战时老员工 / 尾声老员工 | 老员工方向的时代切换 | 靠胡须、发型、衣服、体态和站位区分年代。 |
| 演员5 | 初创青年员工 / 战时青年员工 / 尾声青年员工 | 青年员工方向的时代切换 | 尾声青年员工是后辈角色，不承接第一幕人物年龄。 |
| 演员6 | 负责清点的员工 / 第二幕工厂员工 / 设备清点员工 | 信息递送和清单功能 | 建议相对固定为“资料功能演员”，台词克制，避免抢主线。 |
| 演员10 | 搬运员工 / 机修员工 / 重庆厂员工 | 动作和复工场面 | 以身体调度为主，必要时补一句技术或现场短句。 |

若演员人数进一步压缩，优先保留演员4、演员5、演员6三条员工功能线。搬运、机修和重庆厂动作可由演员4、演员5、演员10与无台词群像共同完成。

## 五、台本阶段写法提醒

台本拆分时，建议在角色名后同时标注演员编号，例如：

- 初创员工（演员4）
- 战时老员工（演员4分饰）
- 尾声老员工（演员4分饰）
- 初创青年员工（演员5）
- 战时青年员工（演员5分饰）
- 尾声青年员工（演员5分饰）
- 清点员工/文书员工/设备清点员工（演员6）

这样可以同时满足两件事：一是让导演组看清10人以内如何完成调度；二是让文本层面保持角色身份清晰，避免观众误以为同一个普通员工跨越几十年参与所有历史节点。
